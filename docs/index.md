## How to use this reference

- **Search** is full-text and indexes Japanese — type a kanji, grammar form, or English term
- **Tags** on each card page let you filter by level, register, and context
- **Grammar Stacking** chains show how each layer builds meaning from the inside out
- **Anchor Chunks** mark the most reusable phrases worth active recall training
