# The Particle で — Action, Means, and Domain

## From "by bus" to "in the world" to "that's everything"

---

## JLPT Level {#jlpt-level}

!!! abstract "JLPT Level — N5 Foundation through N3 Extended"
    The instrumental and locational で is N5 — among the first particles learned. But the scope/totalizing use (<ruby>範囲<rt>はんい</rt></ruby>・<ruby>限界<rt>げんかい</rt></ruby>) — superlatives, sums, and boundaries — is often undertaught until N4, where it suddenly appears everywhere in descriptions, comparisons, and accounting. Native speakers feel these as one unified particle; learners often fragment them.

    | Tier | Pattern | Description | Example |
    |------|---------|-------------|---------|
    | N5 — Foundation | Means / instrument | Tool, method, vehicle | バスで<ruby>行<rt>い</rt></ruby>く。 |
    | N5 — Foundation | Location of action | Place where an action occurs | <ruby>図書<rt>としょ</rt></ruby><ruby>館<rt>かん</rt></ruby>で<ruby>勉強<rt>べんきょう</rt></ruby>する。 |
    | **N4 — Core ← You are here** | **Scope / totalizing** | **Domain of evaluation, sum total, boundary** | <ruby>日本<rt>にっぽん</rt></ruby>で<ruby>一番<rt>いちばん</rt></ruby><ruby>高<rt>たか</rt></ruby>い。<ruby>全部<rt>ぜんぶ</rt></ruby>で<ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>。 |
    | N4 — Foundation | Temporal limit | Time within which something completes | <ruby>一<rt>いち</rt></ruby><ruby>時間<rt>じかん</rt></ruby>で<ruby>終<rt>お</rt></ruby>わる。 |
    | N4 — Foundation | Material | Substance something is made from | <ruby>紙<rt>かみ</rt></ruby>で<ruby>作<rt>つく</rt></ruby>る。 |
    | N4 — Foundation | Cause / reason | Reason for a result or state | <ruby>病気<rt>びょうき</rt></ruby>で<ruby>休<rt>やす</rt></ruby>む。 |
    | N3 — Extended | State / manner | Condition in which an action occurs | <ruby>一人<rt>ひとり</rt></ruby>で<ruby>行<rt>い</rt></ruby>く。<ruby>裸<rt>はだか</rt></ruby>で<ruby>寝<rt>ね</rt></ruby>る。 |
    | N3 — Extended | Company | Acting as a group | <ruby>家族<rt>かぞく</rt></ruby>で<ruby>旅行<rt>りょこう</rt></ruby>する。 |

## Quick Summary {#quick-summary}

!!! tip "At a glance"
    - で marks the **context** in which an action happens — whether that context is a physical place (<ruby>図書<rt>としょ</rt></ruby><ruby>館<rt>かん</rt></ruby>で), a tool (ペンで), a domain of evaluation (<ruby>日本<rt>にっぽん</rt></ruby>で<ruby>一番<rt>いちばん</rt></ruby>), a time limit (<ruby>三<rt>みっ</rt></ruby><ruby>日<rt>か</rt></ruby>で), or a total sum (<ruby>全部<rt>ぜんぶ</rt></ruby>で).
    - The core metaphor unifying all uses: **で draws a boundary around the relevant field** — the field of action, the field of comparison, the field of resources, or the field of time.
    - The scope/totalizing use is the focus: で marks the **domain** within which a statement applies (<ruby>日本<rt>にっぽん</rt></ruby>で<ruby>一番<rt>いちばん</rt></ruby>) or the **sum total** that a number represents (<ruby>合計<rt>ごうけい</rt></ruby>で<ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>). English splits these into "in," "for," "by," and "with"; Japanese sees them as one particle.
    - The boundary between "location of action" (<ruby>図書<rt>としょ</rt></ruby><ruby>館<rt>かん</rt></ruby>で) and "scope" (<ruby>日本<rt>にっぽん</rt></ruby>で<ruby>一番<rt>いちばん</rt></ruby>) can feel blurry because both mark a field — but the key distinction is whether the field is a stage for action or a frame for evaluation.

## Core Meaning {#core-meaning}

!!! note "Core Rule"
    Use で to mark the field, means, or boundary within which an action, evaluation, or state is contained. If you can rephrase it as "within the field of X," "by means of X," or "at the limit of X," で is likely the particle.

| バスで<ruby>行<rt>い</rt></ruby>く。 | I'll go by bus [means — the bus is the vehicle/method]. |
|---|---|
| | *The bus is the means of movement. It defines how the going happens.* |

| <ruby>図書<rt>としょ</rt></ruby><ruby>館<rt>かん</rt></ruby>で<ruby>勉強<rt>べんきょう</rt></ruby>する。 | I study at the library [location of action — the library is the stage]. |
|---|---|
| | *The library is the field where studying occurs. It defines where the action happens.* |

| <ruby>日本<rt>にっぽん</rt></ruby>で<ruby>一番<rt>いちばん</rt></ruby><ruby>高<rt>たか</rt></ruby>い<ruby>山<rt>やま</rt></ruby>。 | The tallest mountain in Japan [scope — Japan is the domain of evaluation]. |
|---|---|
| | *Japan is the field within which "tallest" is measured. It defines the comparison set.* |

| <ruby>全部<rt>ぜんぶ</rt></ruby>で<ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>です。 | It's 1000 yen in total [totalizing — the sum is contained within this amount]. |
|---|---|
| | *"Everything" is the scope; 1000 yen is the total within that scope.* |

| <ruby>一<rt>いち</rt></ruby><ruby>時間<rt>じかん</rt></ruby>で<ruby>終<rt>お</rt></ruby>わる。 | It finishes in one hour / within one hour [temporal limit — the hour is the boundary]. |
|---|---|
| | *One hour is the container of time. It defines the limit.* |

| <ruby>病気<rt>びょうき</rt></ruby>で<ruby>学校<rt>がっこう</rt></ruby>を<ruby>休<rt>やす</rt></ruby>んだ。 | I missed school due to illness [cause — illness is the reason]. |
|---|---|
| | *Illness is the causal field. It defines why the absence happened.* |

## The Seven Uses of で {#the-seven-uses-of-de}

### 1. Means / Instrument (<ruby>手段<rt>しゅだん</rt></ruby>・<ruby>方法<rt>ほうほう</rt></ruby>)

The most famous use. で marks the tool, vehicle, method, or medium used to perform an action.

| Pattern | Example |
|---------|---------|
| Vehicle | <ruby>電車<rt>でんしゃ</rt></ruby>で<ruby>通勤<rt>つうきん</rt></ruby>する |
| Tool | <ruby>包丁<rt>ほうちょう</rt></ruby>で<ruby>切<rt>き</rt></ruby>る |
| Method | <ruby>日本<rt>にっぽん</rt></ruby><ruby>語<rt>ご</rt></ruby>で<ruby>話<rt>はな</rt></ruby>す |
| Material (means sense) | <ruby>手<rt>て</rt></ruby>で<ruby>触<rt>さわ</rt></ruby>る |
| Language / medium | メールで<ruby>連絡<rt>れんらく</rt></ruby>する |

!!! note "Means で is active"
    The means is something the agent actively uses. It is not the material something is made of (that's a different use), nor the company (that's another use).

### 2. Location of Action (<ruby>動作<rt>どうさ</rt></ruby>の<ruby>場所<rt>ばしょ</rt></ruby>)

で marks the place where an action occurs — as opposed to に, which marks the place where something exists or the target of movement.

| Pattern | Example |
|---------|---------|
| Action stage | <ruby>教室<rt>きょうしつ</rt></ruby>で<ruby>授業<rt>じゅぎょう</rt></ruby>を<ruby>受<rt>う</rt></ruby>ける |
| Event venue | <ruby>公園<rt>こうえん</rt></ruby>でコンサートがある |
| Work location | <ruby>会社<rt>かいしゃ</rt></ruby>で<ruby>働<rt>はたら</rt></ruby>く |
| Eating location | レストランで<ruby>食<rt>た</rt></ruby>べる |

!!! warning "で vs に for location"
    This is the classic N5 struggle. See the dedicated section below.

### 3. Scope / Domain (<ruby>範囲<rt>はんい</rt></ruby>) — THE FOCUS

で marks the field within which an evaluation, comparison, or ranking applies. This is the "scope" use.

| Sub-type | Pattern | Example |
|----------|---------|---------|
| Superlative domain | 〜で<ruby>一番<rt>いちばん</rt></ruby> | クラスで<ruby>一番<rt>いちばん</rt></ruby><ruby>頭<rt>あたま</rt></ruby>がいい |
| Comparative domain | 〜で〜の<ruby>中<rt>なか</rt></ruby>で | <ruby>日本<rt>にっぽん</rt></ruby>で<ruby>有名<rt>ゆうめい</rt></ruby>な<ruby>料理<rt>りょうり</rt></ruby> |
| Group evaluation | 〜で | チームで<ruby>最<rt>さい</rt></ruby><ruby>年少<rt>ねんしょう</rt></ruby> |
| Exhaustive scope | これで / それで | これで<ruby>全部<rt>ぜんぶ</rt></ruby>です |

The particle says: "Within the boundary of this set, the following evaluation holds."

| <ruby>世界<rt>せかい</rt></ruby>で<ruby>二<rt>に</rt></ruby><ruby>番<rt>ばん</rt></ruby><ruby>目<rt>め</rt></ruby>に<ruby>大<rt>おお</rt></ruby>きい。 | The second largest in the world. |
|---|---|
| | *The world is the evaluation set. で contains the superlative.* |

| うちの<ruby>会社<rt>かいしゃ</rt></ruby>で<ruby>一番<rt>いちばん</rt></ruby><ruby>若<rt>わか</rt></ruby>い。 | The youngest in our company. |
|---|---|
| | *The company is the comparison group.* |

### 4. Totalizing / Summation (<ruby>合計<rt>ごうけい</rt></ruby>・<ruby>総計<rt>そうけい</rt></ruby>)

A specialized scope use. で marks the total sum or complete set.

| Pattern | Example |
|---------|---------|
| Total sum | <ruby>全部<rt>ぜんぶ</rt></ruby>で<ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby> |
| Per-unit | <ruby>一人<rt>ひとり</rt></ruby>で<ruby>五百<rt>ごひゃく</rt></ruby><ruby>円<rt>えん</rt></ruby> |
| Aggregate | <ruby>合計<rt>ごうけい</rt></ruby>で<ruby>十<rt>じゅう</rt></ruby><ruby>人<rt>にん</rt></ruby> |
| Exhaustive | これで<ruby>終<rt>お</rt></ruby>わり |

!!! note "Totalizing で as 'in total'"
    English uses "in total," "altogether," or "for" in these contexts. Japanese uses で because the total amount is the field that contains all the individual items.

| <ruby>家族<rt>かぞく</rt></ruby><ruby>全部<rt>ぜんぶ</rt></ruby>で<ruby>五<rt>ご</rt></ruby><ruby>人<rt>にん</rt></ruby>です。 | There are five people in the family altogether. |
|---|---|
| | *The family is the set; five is the total within it.* |

| これで<ruby>十分<rt>じゅうぶん</rt></ruby>です。 | This is enough [this covers the requirement]. |
|---|---|
| | *"This" marks the boundary of sufficiency. It contains everything needed.* |

### 5. Temporal Limit (<ruby>期限<rt>きげん</rt></ruby>・<ruby>時間<rt>じかん</rt></ruby><ruby>的<rt>てき</rt></ruby><ruby>限界<rt>げんかい</rt></ruby>)

で marks the time within which something is completed or the deadline/boundary of an action.

| Pattern | Example |
|---------|---------|
| Completion time | <ruby>三<rt>みっ</rt></ruby><ruby>日<rt>か</rt></ruby>でできる |
| Duration limit | <ruby>一<rt>いち</rt></ruby><ruby>時間<rt>じかん</rt></ruby>で<ruby>終<rt>お</rt></ruby>わる |
| Deadline | <ruby>今日<rt>きょう</rt></ruby>で<ruby>締<rt>し</rt></ruby>め<ruby>切<rt>き</rt></ruby>り |
| Point of change | <ruby>来月<rt>らいげつ</rt></ruby>で<ruby>三<rt>さん</rt></ruby><ruby>年<rt>ねん</rt></ruby><ruby>目<rt>め</rt></ruby> |

!!! note "Temporal で vs time に"
    <ruby>三<rt>さん</rt></ruby><ruby>時<rt>じ</rt></ruby>に<ruby>終<rt>お</rt></ruby>わる = ends at 3:00 (point in time). <ruby>三<rt>さん</rt></ruby><ruby>時間<rt>じかん</rt></ruby>で<ruby>終<rt>お</rt></ruby>わる = ends within 3 hours (duration limit). で marks the container of time; に marks the point on the timeline.

### 6. Material (<ruby>材料<rt>ざいりょう</rt></ruby>)

で marks the raw material something is made from. This overlaps with means but is distinct: material is consumed or transformed, not just used.

| Pattern | Example |
|---------|---------|
| Raw material | <ruby>木<rt>き</rt></ruby>で<ruby>作<rt>つく</rt></ruby>る |
| Ingredient | <ruby>小麦<rt>こむぎ</rt></ruby><ruby>粉<rt>こな</rt></ruby>で<ruby>作<rt>つく</rt></ruby>るパン |
| Resource | <ruby>水<rt>みず</rt></ruby>で<ruby>発電<rt>はつでん</rt></ruby>する |

!!! note "Material で vs から"
    See the comparison section below.

### 7. Cause / Reason (<ruby>原因<rt>げんいん</rt></ruby>・<ruby>理由<rt>りゆう</rt></ruby>)

で marks the cause of a result, especially natural causes, reasons for absence, or causal conditions.

| Pattern | Example |
|---------|---------|
| Natural cause | <ruby>地震<rt>じしん</rt></ruby>で<ruby>建物<rt>たてもの</rt></ruby>が<ruby>倒<rt>たお</rt></ruby>れた |
| Reason for absence | <ruby>風邪<rt>かぜ</rt></ruby>で<ruby>休<rt>やす</rt></ruby>んだ |
| Causal condition | <ruby>不<rt>ふ</rt></ruby><ruby>注意<rt>ちゅうい</rt></ruby>で<ruby>怪我<rt>けが</rt></ruby>をした |

!!! note "Cause で vs から / ので"
    で is more compact and often implies direct causation. から is explanatory; ので is softer and more polite.

### 8. State / Manner / Company (<ruby>様態<rt>ようたい</rt></ruby>・<ruby>同行<rt>どうこう</rt></ruby>)

で marks the condition or group configuration in which an action occurs.

| Pattern | Example |
|---------|---------|
| Alone | <ruby>一人<rt>ひとり</rt></ruby>で<ruby>行<rt>い</rt></ruby>く |
| As a group | <ruby>家族<rt>かぞく</rt></ruby>で<ruby>旅行<rt>りょこう</rt></ruby>する |
| In a state | <ruby>無料<rt>むりょう</rt></ruby>で<ruby>入場<rt>にゅうじょう</rt></ruby>する |
| Bare/naked state | <ruby>裸<rt>はだか</rt></ruby>で<ruby>走<rt>はし</rt></ruby>る |

!!! note "<ruby>一人<rt>ひとり</rt></ruby>で as scope and means"
    <ruby>一人<rt>ひとり</rt></ruby>で can mean "by oneself" (means) or "alone" (state). The underlying logic is the same: one person is the field containing the action.

## The Scope/Totalizing Deep Dive {#the-scopetotalizing-deep-dive}

This is the use most learners encounter late and understand incompletely. It deserves focused treatment.

### Evaluative Scope: Superlatives and Rankings

When you evaluate something as "the most X" or "the best Y," you need a domain. で marks that domain.

| Domain | Evaluation | Full Phrase |
|--------|-----------|-------------|
| <ruby>日本<rt>にっぽん</rt></ruby> | <ruby>一番<rt>いちばん</rt></ruby><ruby>高<rt>たか</rt></ruby>い | <ruby>日本<rt>にっぽん</rt></ruby>で<ruby>一番<rt>いちばん</rt></ruby><ruby>高<rt>たか</rt></ruby>いビル |
| クラス | <ruby>一番<rt>いちばん</rt></ruby><ruby>背<rt>せ</rt></ruby>が<ruby>高<rt>たか</rt></ruby>い | クラスで<ruby>一番<rt>いちばん</rt></ruby><ruby>背<rt>せ</rt></ruby>が<ruby>高<rt>たか</rt></ruby>い |
| <ruby>世界<rt>せかい</rt></ruby> | <ruby>二<rt>に</rt></ruby><ruby>番<rt>ばん</rt></ruby><ruby>目<rt>め</rt></ruby>に<ruby>大<rt>おお</rt></ruby>きい | <ruby>世界<rt>せかい</rt></ruby>で<ruby>二<rt>に</rt></ruby><ruby>番<rt>ばん</rt></ruby><ruby>目<rt>め</rt></ruby>に<ruby>大<rt>おお</rt></ruby>きい<ruby>国<rt>くに</rt></ruby> |
| チーム | <ruby>最<rt>さい</rt></ruby><ruby>年少<rt>ねんしょう</rt></ruby> | チームで<ruby>最<rt>さい</rt></ruby><ruby>年少<rt>ねんしょう</rt></ruby>の<ruby>選手<rt>せんしゅ</rt></ruby> |
| うち | <ruby>一番<rt>いちばん</rt></ruby><ruby>古<rt>ふる</rt></ruby>い | うちで<ruby>一番<rt>いちばん</rt></ruby><ruby>古<rt>ふる</rt></ruby>い<ruby>車<rt>くるま</rt></ruby> |

!!! warning "English 'in' = Japanese で"
    English uses "in" for these: "the tallest in the class." Japanese uses で, not に. This is because the class is the field of evaluation, not a physical location where something exists. に marks existence; で marks evaluative domains.

| クラスで<ruby>一番<rt>いちばん</rt></ruby><ruby>頭<rt>あたま</rt></ruby>がいい。 | The smartest in the class. |
|---|---|
| | *で marks the class as the comparison set. に would imply the intelligence exists inside the classroom as a physical location — wrong metaphor.* |

### Totalizing Scope: Sums and Aggregates

で marks the total amount that contains all individual parts.

| Pattern | Example | Meaning |
|---------|---------|---------|
| <ruby>全部<rt>ぜんぶ</rt></ruby>で | <ruby>全部<rt>ぜんぶ</rt></ruby>で<ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby> | 1000 yen altogether |
| <ruby>合計<rt>ごうけい</rt></ruby>で | <ruby>合計<rt>ごうけい</rt></ruby>で<ruby>五<rt>ご</rt></ruby><ruby>人<rt>にん</rt></ruby> | 5 people total |
| <ruby>一人<rt>ひとり</rt></ruby>で | <ruby>一人<rt>ひとり</rt></ruby>で<ruby>三百<rt>さんびゃく</rt></ruby><ruby>円<rt>えん</rt></ruby> | 300 yen per person / 300 yen for one person |
| <ruby>合<rt>あ</rt></ruby>わせてで | (rare) | — |
| <ruby>総額<rt>そうがく</rt></ruby>で | <ruby>総額<rt>そうがく</rt></ruby>で<ruby>百<rt>ひゃく</rt></ruby><ruby>万<rt>まん</rt></ruby><ruby>円<rt>えん</rt></ruby> | 1 million yen total |
| これで | これで<ruby>全部<rt>ぜんぶ</rt></ruby>です | That's everything |

!!! note "これで as scope closure"
    これで<ruby>終<rt>お</rt></ruby>わり doesn't just mean "It ends with this" (means). It means "This covers the scope — we are done." The で marks the boundary of completion.

| これで<ruby>十分<rt>じゅうぶん</rt></ruby>だ。 | This is enough [this covers all requirements]. |
|---|---|
| | *The scope of "enough" is defined by "this." で contains the sufficiency.* |

### Limiting Scope: Deadlines and Boundaries

で can mark the point at which something stops or the boundary is reached.

| Pattern | Example | Meaning |
|---------|---------|---------|
| Time boundary | <ruby>今日<rt>きょう</rt></ruby>で<ruby>辞<rt>や</rt></ruby>める | Quit as of today |
| Duration boundary | <ruby>一<rt>いち</rt></ruby><ruby>週間<rt>しゅうかん</rt></ruby>で<ruby>終<rt>お</rt></ruby>わる | Finish within a week |
| Final item | これで<ruby>最後<rt>さいご</rt></ruby> | This is the last one |
| Limit | ここでやめる | Stop here |

| <ruby>来月<rt>らいげつ</rt></ruby>で<ruby>三<rt>さん</rt></ruby><ruby>年<rt>ねん</rt></ruby><ruby>目<rt>め</rt></ruby>になる。 | It will become the third year as of next month. |
|---|---|
| | *Next month is the boundary. The state of "third year" begins at the edge of that month.* |

## How It Works: The Field Metaphor {#how-it-works}

All uses of で can be unified under a single spatial metaphor: **the particle draws a circle around a field, and the action or evaluation happens inside that circle.**

| Use | Field Type | Metaphor |
|-----|-----------|----------|
| Means | Method field | "Within the method of [bus], movement occurs" |
| Location | Physical field | "Within the space of [library], studying occurs" |
| Scope | Evaluative field | "Within the set of [Japan], this is the tallest" |
| Total | Sum field | "Within the total of [1000 yen], everything is contained" |
| Time limit | Temporal field | "Within the boundary of [1 hour], completion occurs" |
| Cause | Causal field | "Within the condition of [illness], absence occurs" |
| State | Conditional field | "Within the state of [alone], action occurs" |

This is why a single particle covers such apparently different concepts. Japanese treats them as one grammatical gesture: **containment within a relevant boundary.**

## Japanese Mindset {#japanese-mindset}

English slices the world finely: "by bus" (means), "at the library" (location), "in Japan" (scope), "in one hour" (time), "from paper" (material), "due to illness" (cause). Japanese sees a pattern: all of these are **fields that contain the action**.

The Japanese speaker doesn't ask "What kind of relationship is this?" but rather "What is the containing field?" If the library contains the studying, it's で. If the class contains the comparison, it's で. If one hour contains the completion, it's で.

- **Containment over relationship.** English particles often mark grammatical relationships (agent, patient, instrument). Japanese particles often mark containers (space, time, domain, means). で is the ultimate container particle.
- **Scope as natural evaluation.** English needs "in" or "of" for superlatives (tallest in the world). Japanese uses the same particle it uses for buses and libraries. The world is just a bigger container.
- **Total as closed field.** <ruby>全部<rt>ぜんぶ</rt></ruby>で<ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby> treats the total as a closed circle: everything fits inside 1000 yen. The particle closes the set.

## Nuance Comparison {#nuance-comparison}

### で vs に (Location)

The most important distinction for learners.

| Particle | Function | Example | English |
|----------|----------|---------|---------|
| に | Existence / target / destination | <ruby>図書<rt>としょ</rt></ruby><ruby>館<rt>かん</rt></ruby>にある | It is in the library |
| に | Target of movement | <ruby>図書<rt>としょ</rt></ruby><ruby>館<rt>かん</rt></ruby>に<ruby>行<rt>い</rt></ruby>く | Go to the library |
| に | Time point | <ruby>三<rt>さん</rt></ruby><ruby>時<rt>じ</rt></ruby>に<ruby>行<rt>い</rt></ruby>く | Go at 3:00 |
| で | Action stage | <ruby>図書<rt>としょ</rt></ruby><ruby>館<rt>かん</rt></ruby>で<ruby>勉強<rt>べんきょう</rt></ruby>する | Study at the library |
| で | Scope | <ruby>図書<rt>としょ</rt></ruby><ruby>館<rt>かん</rt></ruby>で<ruby>一番<rt>いちばん</rt></ruby><ruby>静<rt>しず</rt></ruby>か | The quietest in the library |

!!! note "The test"
    If you can insert "exist" and it makes sense, use に: <ruby>図書<rt>としょ</rt></ruby><ruby>館<rt>かん</rt></ruby>に<ruby>本<rt>ほん</rt></ruby>がある (books exist in the library). If you can insert "do something" and it makes sense, use で: <ruby>図書<rt>としょ</rt></ruby><ruby>館<rt>かん</rt></ruby>で<ruby>勉強<rt>べんきょう</rt></ruby>する (study at the library).

| レストランで<ruby>食<rt>た</rt></ruby>べる。 | Eat at the restaurant [action stage]. |
|---|---|
| レストランにある。 | It is in the restaurant [existence]. |

### で vs から (Material)

Both can mean "made from," but the nuance differs.

| Particle | Nuance | Example |
|----------|--------|---------|
| で | Material is obvious / processed | <ruby>紙<rt>かみ</rt></ruby>で<ruby>作<rt>つく</rt></ruby>った<ruby>箱<rt>ばこ</rt></ruby> (a box made of paper — paper is the material) |
| から | Material is transformed / origin | <ruby>葡萄<rt>ぶどう</rt></ruby>から<ruby>作<rt>つく</rt></ruby>ったワイン (wine made from grapes — transformation) |
| で | General material | <ruby>木<rt>き</rt></ruby>で<ruby>作<rt>つく</rt></ruby>る |
| から | Raw material with change | <ruby>牛乳<rt>ぎゅうにゅう</rt></ruby>からチーズを<ruby>作<rt>つく</rt></ruby>る |

!!! tip "The distinction is blurry"
    In practice, で and から overlap for material. で is more common for everyday objects; から is more common for derived products.

### で vs から / ので (Cause)

| Particle | Register / Nuance | Example |
|----------|-------------------|---------|
| で | Compact, direct cause | <ruby>病気<rt>びょうき</rt></ruby>で<ruby>休<rt>やす</rt></ruby>む |
| から | Explanatory, logical | <ruby>病気<rt>びょうき</rt></ruby>だから<ruby>休<rt>やす</rt></ruby>む |
| ので | Soft, polite | <ruby>病気<rt>びょうき</rt></ruby>なので<ruby>休<rt>やす</rt></ruby>みます |
| ために | Formal, objective | <ruby>病気<rt>びょうき</rt></ruby>のために<ruby>休<rt>やす</rt></ruby>む |

### で vs も (Scope/Additive)

| Particle | Function | Example |
|----------|----------|---------|
| で | Scope of evaluation | <ruby>日本<rt>にっぽん</rt></ruby>で<ruby>一番<rt>いちばん</rt></ruby> |
| も | Additive inclusion | <ruby>日本<rt>にっぽん</rt></ruby>も<ruby>行<rt>い</rt></ruby>く |

!!! warning "Don't confuse these"
    <ruby>日本<rt>にっぽん</rt></ruby>で<ruby>行<rt>い</rt></ruby>く = "Go in Japan" or "Go by Japan" (nonsense). <ruby>日本<rt>にっぽん</rt></ruby>も<ruby>行<rt>い</rt></ruby>く = "I'll go to Japan too."

## Register and Formality {#register-and-formality}

で is neutral and appears at every register. Formality shifts come from the main predicate, not from the particle.

=== ":material-chat-outline: Casual"
    バスで<ruby>行<rt>い</rt></ruby>く。<ruby>全部<rt>ぜんぶ</rt></ruby>で<ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>だよ。

    *"I'll go by bus. It's 1000 yen total."*

    **Shift:** Plain form. で is unchanged.

=== ":material-tie: Basic Polite"
    バスで<ruby>参<rt>まい</rt></ruby>ります。<ruby>全部<rt>ぜんぶ</rt></ruby>で<ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>です。

    *"I will go by bus. It is 1000 yen in total."*

    **Shift:** です／ます. で remains で.

=== ":material-briefcase-outline: Extra Polite"
    <ruby>電車<rt>でんしゃ</rt></ruby>でお<ruby>伺<rt>うかが</rt></ruby>いいたします。<ruby>合計<rt>ごうけい</rt></ruby>で<ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>でございます。

    *"I will visit by train. The total is 1000 yen."*

    **Shift:** Keigo verbs. で remains で. The particle is register-invariant.

## Common Mistakes {#common-mistakes}

### 1. Using に for evaluative scope

!!! failure "English transfer error"
    <ruby>日本<rt>にっぽん</rt></ruby>に<ruby>一番<rt>いちばん</rt></ruby><ruby>高<rt>たか</rt></ruby>い<ruby>山<rt>やま</rt></ruby>。

    ~に marks existence or direction. For evaluative scope ("in Japan"), で is required.~

!!! success "Correct"
    <ruby>日本<rt>にっぽん</rt></ruby>で<ruby>一番<rt>いちばん</rt></ruby><ruby>高<rt>たか</rt></ruby>い<ruby>山<rt>やま</rt></ruby>。

### 2. Confusing temporal point (に) with temporal limit (で)

!!! failure "Time particle error"
    <ruby>三<rt>さん</rt></ruby><ruby>時間<rt>じかん</rt></ruby>に<ruby>終<rt>お</rt></ruby>わる。

    ~<ruby>三<rt>さん</rt></ruby><ruby>時<rt>じ</rt></ruby>に = at 3:00 (point, に). <ruby>三<rt>さん</rt></ruby><ruby>時間<rt>じかん</rt></ruby>で = within 3 hours (duration limit, で).~

!!! success "Correct"
    <ruby>三<rt>さん</rt></ruby><ruby>時間<rt>じかん</rt></ruby>で<ruby>終<rt>お</rt></ruby>わる。

### 3. Using で for existence location

!!! failure "Existence error"
    <ruby>図書<rt>としょ</rt></ruby><ruby>館<rt>かん</rt></ruby>で<ruby>本<rt>ほん</rt></ruby>がある。

    ~Existence uses に: <ruby>図書<rt>としょ</rt></ruby><ruby>館<rt>かん</rt></ruby>に<ruby>本<rt>ほん</rt></ruby>がある. で marks where an action happens, not where something exists.~

!!! success "Correct"
    <ruby>図書<rt>としょ</rt></ruby><ruby>館<rt>かん</rt></ruby>に<ruby>本<rt>ほん</rt></ruby>がある。

### 4. Confusing scope で with topic は

!!! failure "Redundancy"
    <ruby>日本<rt>にっぽん</rt></ruby>では<ruby>一番<rt>いちばん</rt></ruby><ruby>高<rt>たか</rt></ruby>い。

    ~Adding は is possible (contrastive topic: "in Japan [as opposed to elsewhere]"), but <ruby>日本<rt>にっぽん</rt></ruby>で<ruby>一番<rt>いちばん</rt></ruby><ruby>高<rt>たか</rt></ruby>い is the neutral form. Don't overuse は.~

!!! success "Correct"
    <ruby>日本<rt>にっぽん</rt></ruby>で<ruby>一番<rt>いちばん</rt></ruby><ruby>高<rt>たか</rt></ruby>い。

## Hidden Points {#hidden-points}

- **で can stack with other particles in compound forms.** でも (even if), までも (even until), からも (even from). These are not the particle で alone, but で participates in the compound.
- **<ruby>一人<rt>ひとり</rt></ruby>で has two readings.** It can mean "by oneself" (means: using only one person) or "alone" (state: in the condition of solitude). Context disambiguates, but the underlying container metaphor is the same.
- **これで can mean "that's why" in some contexts.** When used as a sentence opener (これで<ruby>分<rt>わ</rt></ruby>かった), it means "Now I get it" — the で marks the current moment as the field of understanding. This is related to scope: "within this, comprehension occurs."
- **で can mark the price of a transaction.** <ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>で<ruby>買<rt>か</rt></ruby>った ("bought it for 1000 yen"). This is a specialized scope/totalizing use — the 1000 yen is the field containing the transaction.
- **<ruby>全部<rt>ぜんぶ</rt></ruby>で vs <ruby>全部<rt>ぜんぶ</rt></ruby>が.** <ruby>全部<rt>ぜんぶ</rt></ruby>で<ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby> = "1000 yen total." <ruby>全部<rt>ぜんぶ</rt></ruby>が<ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby> = "Everything is 1000 yen [each]." The particle choice completely changes the math.
- **で can be used for "at the age of."** <ruby>二十歳<rt>はたち</rt></ruby>で<ruby>結婚<rt>けっこん</rt></ruby>した ("Got married at 20"). This is a temporal boundary use — 20 is the field containing the marriage event.
- **The "company" use is culturally significant.** <ruby>家族<rt>かぞく</rt></ruby>で<ruby>旅行<rt>りょこう</rt></ruby>する doesn't just mean "travel with family" — it means "travel as a family unit." The で marks the family as the containing social field. This is why <ruby>一人<rt>ひとり</rt></ruby>で can sound lonely in Japanese: it explicitly marks the absence of a containing group.

## Learner Stage {#learner-stage}

| Stage | What It Looks Like |
|-------|-------------------|
| **Now — N5** | Use で correctly for means and location of action; distinguish で from に for location; avoid using に for evaluative scope |
| **Developing — N5→N4** | Deploy scope/totalizing で naturally (〜で<ruby>一番<rt>いちばん</rt></ruby>, <ruby>全部<rt>ぜんぶ</rt></ruby>で, これで<ruby>終<rt>お</rt></ruby>わり); use temporal limit で correctly (<ruby>三<rt>さん</rt></ruby><ruby>時間<rt>じかん</rt></ruby>で); handle cause で without overusing から |
| **Automatic — N4→N3** | The container metaphor becomes intuitive. You feel why で works for buses, libraries, Japan, sums, and time limits. You no longer fragment で into "different particles." |
| **Mastery** | You control the full で spectrum and know exactly when it competes with に (existence), から (material/cause), も (additive), and は (topic). You also hear the subtle differences between これで<ruby>終<rt>お</rt></ruby>わり (scope closure) and これで<ruby>終<rt>お</rt></ruby>わる (means completion). |

## See Also {#see-also}

### Same Family

- に — Location of existence, target, time point
- を — Direct object
- から — Origin, raw material, cause
- まで — Until, extent

### Commonly Confused With

- に (location) — existence vs. action
- から (material/cause) — transformation vs. direct cause
- ので (conjunction) — soft cause, different word
- でも (conjunction) — "but," unrelated to particle で
- も (additive) — inclusion vs. scope

### Builds From

- N5 particle foundation — understanding case marking
- Verb transitivity — knowing whether an action needs a stage
- Adjective superlatives — <ruby>一番<rt>いちばん</rt></ruby>, <ruby>最<rt>もっと</rt></ruby>も, etc.
- Counters and numbers — for totalizing use

### Unlocks Next

- でも (particle) — "even if / even [noun]" — conditional extreme
- から (material) — derived products and transformation
- ので (conjunction) — soft causal connection
- ては / では — conditional negative (contraction of て + は)
