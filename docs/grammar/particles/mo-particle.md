# The Particle も — Also, Even, and Everything In Between

## From "me too" to "not even once"

---

## JLPT Level {#jlpt-level}

!!! abstract "JLPT Level — N5 Foundation through N3 Extended"
    The additive も is N5 — one of the first particles learners encounter. But the scalar, emphatic, conditional, and negative-polarity uses spread across N4 and N3, forming a dense web that native speakers navigate effortlessly and learners often underuse.

    | Tier | Pattern | Description | Example |
    |------|---------|-------------|---------|
    | N5 — Foundation | Noun + も | Additive — "also / too" | <ruby>私<rt>わたくし</rt></ruby>も<ruby>行<rt>い</rt></ruby>く。 |
    | N5 — Foundation | AもBも | Listing — "both A and B" | <ruby>猫<rt>ねこ</rt></ruby>も<ruby>犬<rt>いぬ</rt></ruby>も<ruby>好<rt>す</rt></ruby>き。 |
    | N4 — Foundation | Quantity + も | Scalar — "as much as / a whole / not even" | <ruby>三<rt>さん</rt></ruby><ruby>時間<rt>じかん</rt></ruby>も<ruby>待<rt>ま</rt></ruby>った。 |
    | N4 — Foundation | Wh-word + も + negative | Negative polarity — "nothing / nobody / nowhere" | <ruby>何<rt>なん</rt></ruby>も<ruby>食<rt>た</rt></ruby>べない。 |
    | **N4 — Core ← You are here** | **〜ても / 〜でも** | **Conditional — "even if / even though"** | <ruby>高<rt>たか</rt></ruby>くても<ruby>買<rt>か</rt></ruby>う。 |
    | N3 — Extended | Noun + でも | Extreme — "even [noun]" | <ruby>子供<rt>こども</rt></ruby>でもできる。 |
    | N3 — Extended | 〜もしない / 〜もある | Minimal / maximal — "not even / there is also" | <ruby>一<rt>いち</rt></ruby><ruby>度<rt>ど</rt></ruby>も<ruby>行<rt>い</rt></ruby>かない。 |
    | N2 — Awareness | 〜も〜ば〜も | Correlative — "both… and… / neither… nor…" | <ruby>行<rt>い</rt></ruby>くも<ruby>行<rt>い</rt></ruby>かないも<ruby>自由<rt>じゆう</rt></ruby>だ。 |

## Quick Summary {#quick-summary}

!!! tip "At a glance"
    - も is a **multipurpose particle** that replaces は, が, or を on a noun to add it to a set. <ruby>私<rt>わたくし</rt></ruby>は<ruby>行<rt>い</rt></ruby>く → <ruby>私<rt>わたくし</rt></ruby>も<ruby>行<rt>い</rt></ruby>く ("I too will go").
    - Beyond simple addition, も scales: attached to quantities it means "as much as / a whole" (<ruby>三<rt>さん</rt></ruby><ruby>時間<rt>じかん</rt></ruby>も); attached to minimal quantities with negative verbs it means "not even" (<ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>もない).
    - With wh-words (<ruby>何<rt>なん</rt></ruby>, <ruby>誰<rt>だれ</rt></ruby>, どこ) and negative verbs, も creates universal negatives: <ruby>何<rt>なん</rt></ruby>も<ruby>食<rt>た</rt></ruby>べない = "I eat nothing."
    - With the te-form (〜ても / 〜でも), も creates conditional extremes: "even if / even though."
    - With bare nouns in extreme contexts, も means "even": <ruby>子供<rt>こども</rt></ruby>でもできる = "Even a child can do it."
    - The core logic unifying all uses: も marks an item as **at the edge of a set** — either adding to the set (additive), pushing the boundary of the set (scalar), or exhausting the set entirely (universal / negative polarity).

## Core Meaning {#core-meaning}

!!! note "Core Rule"
    Use も when you want to mark an item as belonging to a set in a way that emphasizes its inclusion, its extremity, or its exhaustiveness. The particle absorbs and replaces whatever case particle (は, が, を) would normally mark that item.

| <ruby>私<rt>わたくし</rt></ruby>も<ruby>行<rt>い</rt></ruby>く。 | I will go too [additive — I am added to the set of goers]. |
|---|---|
| | *も replaces が or は. The speaker joins an existing group.* |

| <ruby>三<rt>さん</rt></ruby><ruby>時間<rt>じかん</rt></ruby>も<ruby>待<rt>ま</rt></ruby>った。 | I waited a whole three hours [scalar — the quantity is emphasized as large]. |
|---|---|
| | *も attaches to the quantity. The three hours are marked as notable, burdensome, or impressive.* |

| <ruby>何<rt>なん</rt></ruby>も<ruby>食<rt>た</rt></ruby>べない。 | I eat nothing [negative polarity — the wh-word + も exhausts the set of possible objects]. |
|---|---|
| | *も attaches to <ruby>何<rt>なん</rt></ruby> in a negative context, creating universal negation.* |

| <ruby>高<rt>たか</rt></ruby>くても<ruby>買<rt>か</rt></ruby>う。 | Even if it's expensive, I'll buy it [conditional extreme]. |
|---|---|
| | *も attaches to the te-form. The condition (expensive) is marked as extreme but insufficient to prevent the action.* |

| <ruby>子供<rt>こども</rt></ruby>でもできる。 | Even a child can do it [extreme — the noun is at the low end of capability]. |
|---|---|
| | *も attaches to a noun directly (or to the copula contraction でも). The child represents the minimal expected capability.* |

## The Six Functions of も {#the-six-functions-of-も}

### 1. Additive も — "Also / Too / As Well"

The foundational use. も replaces は, が, or を on a noun to include it in a set already under discussion.

| Pattern | Example | Meaning |
|---------|---------|---------|
| Noun + も | <ruby>私<rt>わたくし</rt></ruby>も。 | Me too. |
| Noun + も + predicate | <ruby>彼<rt>かれ</rt></ruby>も<ruby>来<rt>き</rt></ruby>た。 | He came too. |
| Noun + も + particle + predicate | <ruby>東京<rt>とうきょう</rt></ruby>にも<ruby>行<rt>おこな</rt></ruby>った。 | I went to Tokyo too. |
| Noun + も + noun + も | <ruby>猫<rt>ねこ</rt></ruby>も<ruby>犬<rt>いぬ</rt></ruby>も<ruby>好<rt>す</rt></ruby>き。 | I like both cats and dogs. |

!!! warning "も absorbs particles"
    You cannot double-mark. <ruby>私<rt>わたくし</rt></ruby>はも and <ruby>私<rt>わたくし</rt></ruby>もは are both ungrammatical. も replaces the topic/subject/object marker entirely. For other particles (に, で, から, etc.), も attaches after them: <ruby>東京<rt>とうきょう</rt></ruby>にも, <ruby>学校<rt>がっこう</rt></ruby>でも.

| <ruby>彼<rt>かれ</rt></ruby>にも<ruby>言<rt>い</rt></ruby>った。 | I told him too [に + も]. |
|---|---|
| | *The に marks the recipient; the も adds him to the set of people told.* |

| <ruby>昨日<rt>きのう</rt></ruby>も<ruby>雨<rt>あめ</rt></ruby>だった。 | It rained yesterday too [temporal + も]. |
|---|---|
| | *も attaches to time expressions to add them to a temporal set.* |

### 2. Scalar も — "As Much As / A Whole / Not Even"

When も attaches to quantities, it pushes them to the edge of expectation. The direction (large or small) depends on context and polarity.

**Large / Surprising:**
| <ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>もした。 | It cost as much as 1000 yen [surprisingly expensive]. |
|---|---|
| <ruby>三<rt>さん</rt></ruby><ruby>時間<rt>じかん</rt></ruby>も<ruby>待<rt>ま</rt></ruby>った。 | I waited a whole three hours [surprisingly long]. |
| <ruby>十<rt>じゅう</rt></ruby><ruby>人<rt>にん</rt></ruby>も<ruby>来<rt>き</rt></ruby>た。 | As many as ten people came [surprisingly many]. |

**Small / Dismissive (with negative or downplaying predicates):**
| <ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>もない。 | I don't even have 1000 yen [dismally little]. |
|---|---|
| <ruby>三<rt>さん</rt></ruby><ruby>時間<rt>じかん</rt></ruby>も<ruby>寝<rt>ね</rt></ruby>ていない。 | I haven't even slept three hours [insufficient]. |
| <ruby>一人<rt>ひとり</rt></ruby>もいなかった。 | There wasn't even one person [universal negative]. |

!!! note "The scalar flip"
    The same quantity + も can mean "as much as" (positive, surprising) or "not even" (negative, dismissive). Context and the main predicate determine which reading applies.

### 3. Wh-word + も — Universal and Negative Polarity

When も attaches to question words (wh-words), it creates universal or exhaustive reference. With negative predicates, this becomes universal negation.

| Wh-word | + Negative | Meaning | + Affirmative | Meaning |
|---------|-----------|---------|---------------|---------|
| <ruby>何<rt>なん</rt></ruby>も | <ruby>何<rt>なん</rt></ruby>も<ruby>食<rt>た</rt></ruby>べない | eat nothing | (rare/non-standard) | — |
| <ruby>誰<rt>だれ</rt></ruby>も | <ruby>誰<rt>だれ</rt></ruby>も<ruby>来<rt>こ</rt></ruby>なかった | nobody came | (rare) | — |
| どこも | どこも<ruby>行<rt>い</rt></ruby>かない | go nowhere | どこも<ruby>素敵<rt>すてき</rt></ruby>だ | everywhere is lovely |
| いつも | (rare with neg) | — | いつも<ruby>忙<rt>いそが</rt></ruby>しい | always busy |
| どれも | どれも<ruby>好<rt>す</rt></ruby>きじゃない | none of them liked | どれも<ruby>美味<rt>おい</rt></ruby>しい | all of them delicious |
| いくらも | いくらもない | not much at all | いくらもある | there is plenty |

!!! warning "いつも is lexicalized"
    いつも means "always" and is almost always affirmative. It does not follow the negative polarity pattern. For "never," use <ruby>一<rt>いち</rt></ruby><ruby>度<rt>ど</rt></ruby>も〜ない or <ruby>全然<rt>ぜんぜん</rt></ruby>〜ない.

!!! warning "<ruby>何<rt>なん</rt></ruby>も with affirmative verbs is non-standard"
    <ruby>何<rt>なん</rt></ruby>も<ruby>食<rt>た</rt></ruby>べた is ungrammatical in standard Japanese. For "I ate something," use <ruby>何<rt>なん</rt></ruby>か<ruby>食<rt>た</rt></ruby>べた. For "I ate everything," use <ruby>全部<rt>ぜんぶ</rt></ruby><ruby>食<rt>た</rt></ruby>べた or <ruby>何<rt>なん</rt></ruby>でも<ruby>食<rt>た</rt></ruby>べた.

### 4. Conditional ても / でも — "Even If / Even Though"

も attaches to the te-form of verbs, i-adjectives, and the de-form of nouns/na-adjectives to create extreme conditions.

| Type | Form | Example | Meaning |
|------|------|---------|---------|
| Verb (te-form) | 〜ても | <ruby>行<rt>い</rt></ruby>っても<ruby>無駄<rt>むだ</rt></ruby>だ | Even if [you] go, it's useless |
| i-Adjective | 〜くても | <ruby>高<rt>たか</rt></ruby>くても<ruby>買<rt>か</rt></ruby>う | Even if expensive, [I] buy |
| na-Adjective | 〜でも | <ruby>静<rt>しず</rt></ruby>かでも<ruby>聞<rt>き</rt></ruby>こえる | Even if quiet, [I] can hear |
| Noun | 〜でも | <ruby>雨<rt>あめ</rt></ruby>でも<ruby>出<rt>で</rt></ruby>かける | Even if rain / even rain, [I] go out |
| Negative verb | 〜なくても | <ruby>行<rt>い</rt></ruby>かなくてもいい | Even if [you] don't go, it's fine |

!!! note "でも has three faces"
    1. **Conditional even:** <ruby>雨<rt>あめ</rt></ruby>でも<ruby>出<rt>で</rt></ruby>かける (even if rain)
    2. **Extreme even:** <ruby>子供<rt>こども</rt></ruby>でもできる (even a child)
    3. **Sentence-initial but:** でも、<ruby>高<rt>たか</rt></ruby>いよ (but, it's expensive) — this is a different word entirely, a contraction of それでも.

### 5. Extreme も — "Even [Noun]"

When も attaches directly to nouns (or noun+copula) without a conditional te/de, it marks the noun as an extreme or unexpected member of a category.

| <ruby>子供<rt>こども</rt></ruby>も<ruby>知<rt>し</rt></ruby>っている。 | Even children know [surprisingly basic]. |
|---|---|
| <ruby>彼女<rt>かのじょ</rt></ruby>も<ruby>来<rt>く</rt></ruby>るって。 | Even she's coming [surprisingly, she too]. |
| <ruby>百<rt>ひゃく</rt></ruby><ruby>円<rt>えん</rt></ruby>もあれば<ruby>足<rt>た</rt></ruby>りる。 | Even 100 yen would be enough [surprisingly little]. |

!!! note "Extreme も vs. Additive も"
    The line is blurry. <ruby>彼女<rt>かのじょ</rt></ruby>も<ruby>来<rt>く</rt></ruby>る can mean "She's coming too" (additive) or "Even she's coming" (surprising extreme). Intonation and context disambiguate. Falling intonation = additive; rising or emphatic intonation = extreme.

### 6. Minimal / Maximal も — 〜もしない / 〜もある

も marks the edge of verbal action — either the absence of even minimal action, or the presence of additional action.

| <ruby>一<rt>いち</rt></ruby><ruby>度<rt>ど</rt></ruby>も<ruby>行<rt>おこな</rt></ruby>ったことがない。 | I've never gone even once [minimal]. |
|---|---|
| <ruby>驚<rt>おどろ</rt></ruby>くこともある。 | There are also surprising things [maximal/additive]. |
| <ruby>何<rt>なん</rt></ruby>もしない。 | Do nothing [universal negative]. |
| <ruby>笑<rt>わら</rt></ruby>うこともできない。 | Can't even laugh [extreme inability]. |

## In Real Life {#in-real-life}

!!! example "In Real Life"
    **Situation:** ユイ and ケン are planning a weekend outing. They've been discussing restaurants.

    &gt; ユイ：<ruby>寿司<rt>すし</rt></ruby>も<ruby>食<rt>た</rt></ruby>べたいな。
    &gt; ケン：<ruby>私<rt>わたくし</rt></ruby>も。でも、<ruby>高<rt>たか</rt></ruby>いよ。
    &gt; ユイ：<ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>もしないよ。ランチなら。
    &gt; ケン：<ruby>子供<rt>こども</rt></ruby>でも<ruby>行<rt>い</rt></ruby>ける？
    &gt; ユイ：<ruby>誰<rt>だれ</rt></ruby>でも<ruby>行<rt>い</rt></ruby>ける。<ruby>何<rt>なん</rt></ruby>も<ruby>心配<rt>しんぱい</rt></ruby>しないで。

    *Yui: "I want to eat sushi too."*
    *Ken: "Me too. But it's expensive."*
    *Yui: "It won't even cost 1000 yen. If it's lunch."*
    *Ken: "Can even children go?"*
    *Yui: "Anyone can go. Don't worry about anything."*

    **The も density:** This short exchange contains five distinct も functions:
    1. **<ruby>寿司<rt>すし</rt></ruby>も** — additive (sushi too, in addition to other foods)
    2. **<ruby>私<rt>わたくし</rt></ruby>も** — additive (me too)
    3. **<ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>もしない** — scalar negative (not even 1000 yen)
    4. **<ruby>子供<rt>こども</rt></ruby>でも** — extreme conditional (even if [you're] a child / even a child)
    5. **<ruby>誰<rt>だれ</rt></ruby>でも** — extreme universal (anyone, no matter who)
    6. **<ruby>何<rt>なん</rt></ruby>も** — negative polarity (nothing)

    **What makes this natural:** Native speakers switch between も functions unconsciously. The particle is a Swiss Army knife that scales from simple addition to universal quantification. The density in this exchange is completely normal.

## Grammar Analysis {#grammar-analysis}

### The Particle Replacement Rule

も does not stack with は, が, or を. It replaces them. For other particles, it stacks after them.

| Normal Particle | + も | Example |
|-----------------|------|---------|
| は (topic) | も (replaces) | <ruby>私<rt>わたくし</rt></ruby>は → <ruby>私<rt>わたくし</rt></ruby>も |
| が (subject) | も (replaces) | <ruby>彼<rt>かれ</rt></ruby>が → <ruby>彼<rt>かれ</rt></ruby>も |
| を (object) | も (replaces) | <ruby>本<rt>ほん</rt></ruby>を → <ruby>本<rt>ほん</rt></ruby>も |
| に (direction) | にも | <ruby>東京<rt>とうきょう</rt></ruby>に → <ruby>東京<rt>とうきょう</rt></ruby>にも |
| で (location/means) | でも | <ruby>学校<rt>がっこう</rt></ruby>で → <ruby>学校<rt>がっこう</rt></ruby>でも |
| から (origin) | からも | <ruby>日本<rt>にっぽん</rt></ruby>から → <ruby>日本<rt>にっぽん</rt></ruby>からも |
| へ (direction) | へも | <ruby>家<rt>いえ</rt></ruby>へ → <ruby>家<rt>いえ</rt></ruby>へも |
| と (with) | とも | <ruby>友達<rt>ともだち</rt></ruby>と → <ruby>友達<rt>ともだち</rt></ruby>とも |
| まで (until) | までも | <ruby>夜<rt>よる</rt></ruby>まで → <ruby>夜<rt>よる</rt></ruby>までも |

!!! warning "でも is overloaded"
    <ruby>学校<rt>がっこう</rt></ruby>でも can mean:
    1. "At school too" (で + も, additive)
    2. "Even at school" (で + も, extreme)
    3. "Even if at school" (conditional, though rare with concrete nouns)
    
    Context resolves the ambiguity.

### Scalar も with Numbers

The scalar function has precise pragmatic rules:

| Pattern | Reading | Example |
|---------|---------|---------|
| Number + counter + も + affirmative | "As many as / a whole" | <ruby>三<rt>さん</rt></ruby><ruby>時間<rt>じかん</rt></ruby>も<ruby>待<rt>ま</rt></ruby>った |
| Number + counter + も + negative | "Not even" | <ruby>三<rt>さん</rt></ruby><ruby>時間<rt>じかん</rt></ruby>も<ruby>寝<rt>ね</rt></ruby>ていない |
| <ruby>一<rt>いち</rt></ruby> + counter + も + negative | "Not even one" | <ruby>一<rt>いち</rt></ruby><ruby>度<rt>ど</rt></ruby>も<ruby>行<rt>い</rt></ruby>かない |
| <ruby>少<rt>すこ</rt></ruby>しも + negative | "Not even a little" | <ruby>少<rt>すこ</rt></ruby>しも<ruby>分<rt>わ</rt></ruby>からない |
| ちっとも + negative | "Not at all" | ちっとも<ruby>知<rt>し</rt></ruby>らない |

!!! note "The number one is special"
    <ruby>一<rt>いち</rt></ruby><ruby>度<rt>ど</rt></ruby>も, <ruby>一人<rt>ひとり</rt></ruby>も, <ruby>一<rt>ひと</rt></ruby>つも with negative verbs create the strongest universal negation. They mean "not even once / not even one person / not even one thing."

### Wh-word + も Grammar

| Wh-word | Negative Form | Meaning | Affirmative Form | Meaning |
|---------|---------------|---------|------------------|---------|
| <ruby>何<rt>なん</rt></ruby>も | <ruby>何<rt>なん</rt></ruby>も〜ない | nothing / not anything | (non-standard) | — |
| <ruby>誰<rt>だれ</rt></ruby>も | <ruby>誰<rt>だれ</rt></ruby>も〜ない | nobody / not anyone | <ruby>誰<rt>だれ</rt></ruby>もが<ruby>知<rt>し</rt></ruby>る | everyone knows (literary) |
| どこも | どこも〜ない | nowhere / not anywhere | どこも<ruby>賑<rt>にぎ</rt></ruby>やか | everywhere is lively |
| いつも | (rare) | — | いつも〜 | always |
| どれも | どれも〜ない | none | どれも〜 | all of them |
| いくらも | いくらも〜ない | not much | いくらも〜 | plenty |

!!! warning "<ruby>何<rt>なん</rt></ruby>か vs. <ruby>何<rt>なん</rt></ruby>も"
    <ruby>何<rt>なん</rt></ruby>か (something) is the affirmative counterpart to <ruby>何<rt>なん</rt></ruby>も (nothing). They are not interchangeable:
    - <ruby>何<rt>なん</rt></ruby>か<ruby>食<rt>た</rt></ruby>べた (I ate something)
    - <ruby>何<rt>なん</rt></ruby>も<ruby>食<rt>た</rt></ruby>べなかった (I ate nothing)

## Japanese Mindset {#japanese-mindset}

English tends to add information through new subjects and verbs: "I went. She went too. He also went." Japanese tends to add information through particle manipulation on a stable predicate: <ruby>私<rt>わたくし</rt></ruby>が<ruby>行<rt>おこな</rt></ruby>った。<ruby>彼女<rt>かのじょ</rt></ruby>も<ruby>行<rt>おこな</rt></ruby>った。<ruby>彼<rt>かれ</rt></ruby>も<ruby>行<rt>おこな</rt></ruby>った。 The predicate stays; the participants accumulate.

This reflects a broader Japanese preference for **stable frames with modular participants**. も is the perfect tool for this — it allows the sentence skeleton to remain while new elements are clipped on.

The scalar uses reveal another preference: **quantities as emotional events**. <ruby>三<rt>さん</rt></ruby><ruby>時間<rt>じかん</rt></ruby>も<ruby>待<rt>ま</rt></ruby>った doesn't just state the duration; it evaluates it. The も turns a measurement into an experience. English requires separate words ("a whole three hours," "as much as three hours") to achieve this; Japanese builds it into the particle.

- **Inclusion as social harmony.** Additive も is the grammar of solidarity. Saying <ruby>私<rt>わたくし</rt></ruby>も is not just information — it's alignment. You are joining the other's frame.
- **Extremity as rhetoric.** Scalar も pushes quantities to the edge of believability. It makes numbers feel large, small, shocking, or pitiful.
- **Exhaustion as negation.** Wh-word + も + negative doesn't just deny — it empties the universe of possibilities. <ruby>何<rt>なん</rt></ruby>もない is stronger than ない because it denies the existence of any possible thing.

## Nuance Comparison {#nuance-comparison}

| Grammar Point | Core Meaning | Emotional / Pragmatic Nuance | Trigger Context | JLPT |
|---|---|---|---|---|
| も (additive) | Also / too | Solidarity, inclusion, joining a set | Adding oneself or others to an established group | N5 |
| も (scalar) | As much as / not even | Emphasis on quantity as surprising or insufficient | Quantities with emotional loading | N4 |
| も (wh-negative) | Nothing / nobody / nowhere | Universal negation; emptiness | Negative sentences with wh-words | N4 |
| ても / でも | Even if / even though | Extreme condition insufficient to prevent action | Conditional sentences pushing boundaries | N4 |
| Noun + でも | Even [noun] | Extreme member of a category | Surprising capability or inclusion | N3 |
| さえ / すら | Even | More extreme than も; often negative or literary | "Not even" with stronger dismissal; literary | N3 |
| だけ | Only | Exclusionary; limits a set | "Nothing but" contexts | N4 |
| しか〜ない | Only [negative frame] | "Nothing except"; more emphatic than だけ | Negative exclusivity | N4 |
| でも (sentence-initial) | But | Contrastive conjunction (different from particle も) | Changing topic, pushing back | N5 |

**も vs. さえ:** <ruby>子供<rt>こども</rt></ruby>でもできる means "Even a child can do it" (surprising but neutral). <ruby>子供<rt>こども</rt></ruby>さえできる means "Even a child can do it" (dismissive of adult difficulty, or emphasizing how easy it is). さえ is sharper and more evaluative.

**も vs. だけ:** <ruby>私<rt>わたくし</rt></ruby>も<ruby>行<rt>い</rt></ruby>く ("I too will go" — inclusive) vs. <ruby>私<rt>わたくし</rt></ruby>だけ<ruby>行<rt>い</rt></ruby>く ("Only I will go" — exclusive). They are semantic opposites but structurally similar in replacing particles.

## Alternatives {#alternatives}

### だけ (only — exclusive)

The opposite of additive も. Limits rather than expands.

| | |
|---|---|
| **Example** | <ruby>私<rt>わたくし</rt></ruby>だけ<ruby>行<rt>い</rt></ruby>く。 |
| **English** | "Only I will go." |
| **Intent** | Exclusion — others are not going |
| **vs. も** | も adds to a set; だけ subtracts from it. |

### さえ (even — extreme)

More emphatic and often more critical than scalar も.

| | |
|---|---|
| **Example** | <ruby>子供<rt>こども</rt></ruby>さえ<ruby>知<rt>し</rt></ruby>っている。 |
| **English** | "Even children know [so why don't you?]" |
| **Intent** | Sharp emphasis; can carry reproach |
| **vs. も** | さえ sounds more pointed. も sounds more neutral or inclusive. |

### しか〜ない (only — negative frame)

Exclusive, like だけ, but grammatically negative.

| | |
|---|---|
| **Example** | <ruby>私<rt>わたくし</rt></ruby>しか<ruby>行<rt>い</rt></ruby>かない。 |
| **English** | "Only I will go / Nobody but me is going." |
| **Intent** | Strong exclusion with negative framing |
| **vs. も** | も is positive/inclusive addition; しか〜ない is negative/exclusive limitation. |

### ても vs. とても (completely different)

!!! warning "Visual similarity trap"
    <ruby>高<rt>たか</rt></ruby>くても (even if expensive) and とても<ruby>高<rt>たか</rt></ruby>い (very expensive) share ても/とても but are unrelated. Do not confuse them.

## Register and Formality {#register-and-formality}

も is neutral and appears across all registers. The formality shifts come from the main predicate, not from も itself.

=== ":material-chat-outline: Casual"
    <ruby>私<rt>わたくし</rt></ruby>も<ruby>行<rt>い</rt></ruby>く。<ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>もしないし。

    *"I'm going too. And it won't even cost 1000 yen."*

    **Shift:** Plain form, additive + scalar. The し adds "and what's more."

=== ":material-tie: Basic Polite"
    <ruby>私<rt>わたくし</rt></ruby>も<ruby>行<rt>い</rt></ruby>きます。<ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>もいたしません。

    *"I will go as well. It will not even amount to 1000 yen."*

    **Shift:** です／ます on the main verbs. も itself is unchanged. しない → いたしません (humble negative of する).

=== ":material-briefcase-outline: Extra Polite"
    <ruby>私<rt>わたくし</rt></ruby>どもも<ruby>参<rt>まい</rt></ruby>ります。いかなる<ruby>場合<rt>ばあい</rt></ruby>でも<ruby>対応<rt>たいおう</rt></ruby>いたします。

    *"We too will attend. We will respond in any and all circumstances."*

    **Shift:** <ruby>私<rt>わたくし</rt></ruby>どもも (humble plural + additive も). いかなる<ruby>場合<rt>ばあい</rt></ruby>でも (literary/extreme conditional + も). も survives even in the most formal registers.

## Common Mistakes {#common-mistakes}

### 1. Double-marking with は or が

!!! failure "Incorrect"
    <ruby>私<rt>わたくし</rt></ruby>はも<ruby>行<rt>い</rt></ruby>く。

    ~も replaces は/が. Stacking them is ungrammatical.~

!!! success "Correct"
    <ruby>私<rt>わたくし</rt></ruby>も<ruby>行<rt>い</rt></ruby>く。

### 2. Using <ruby>何<rt>なん</rt></ruby>も with affirmative verbs

!!! failure "Incorrect"
    <ruby>何<rt>なん</rt></ruby>も<ruby>食<rt>た</rt></ruby>べた。

    ~<ruby>何<rt>なん</rt></ruby>も requires a negative predicate in standard Japanese. For affirmative "something," use <ruby>何<rt>なん</rt></ruby>か. For affirmative "everything," use <ruby>全部<rt>ぜんぶ</rt></ruby> or <ruby>何<rt>なん</rt></ruby>でも.~

!!! success "Correct"
    <ruby>何<rt>なん</rt></ruby>か<ruby>食<rt>た</rt></ruby>べた。 (I ate something.)
    <ruby>何<rt>なん</rt></ruby>も<ruby>食<rt>た</rt></ruby>べなかった。 (I ate nothing.)

### 3. Confusing sentence-initial でも with conditional でも

!!! failure "Ambiguous misread"
    でも、<ruby>高<rt>たか</rt></ruby>いよ。

    ~This is the conjunction "but," unrelated to the particle も. It comes from それでも.~

!!! success "Understanding"
    <ruby>雨<rt>あめ</rt></ruby>でも<ruby>出<rt>で</rt></ruby>かける = "Even if it rains, I'll go out" (conditional particle でも).
    でも、<ruby>高<rt>たか</rt></ruby>いよ = "But it's expensive" (sentence-initial conjunction).

### 4. Using いつも with negative verbs for "never"

!!! failure "Pragmatically odd"
    いつも<ruby>行<rt>い</rt></ruby>かない。

    ~This means "I always don't go" (habitual non-attendance), not "I never go." For "never," use <ruby>一<rt>いち</rt></ruby><ruby>度<rt>ど</rt></ruby>も<ruby>行<rt>い</rt></ruby>かない or <ruby>全然<rt>ぜんぜん</rt></ruby><ruby>行<rt>い</rt></ruby>かない.~

!!! success "Correct"
    <ruby>一<rt>いち</rt></ruby><ruby>度<rt>ど</rt></ruby>も<ruby>行<rt>い</rt></ruby>かない。 (I never go — not even once.)

### 5. Assuming scalar も always means "a lot"

!!! failure "Context error"
    <ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>もある。 = "I have as much as 1000 yen" [surprisingly much].
    <ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>もない。 = "I don't even have 1000 yen" [dismally little].

    ~The same scalar も flips meaning based on predicate polarity. Learners sometimes miss the flip.~

## Hidden Points {#hidden-points}

- **も can stack on multiple items in a single predicate.** <ruby>猫<rt>ねこ</rt></ruby>も<ruby>犬<rt>いぬ</rt></ruby>も<ruby>鳥<rt>とり</rt></ruby>も<ruby>好<rt>す</rt></ruby>き is perfectly natural — three も-marked items with one predicate. English would need "I like cats, dogs, and birds" or "I like cats and dogs and even birds."
- **も replaces を, but the object reading remains.** When you say <ruby>本<rt>ほん</rt></ruby>も<ruby>読<rt>よ</rt></ruby>む, the も tells you the book is added to a set, but the underlying grammatical role (object) is unchanged. This is why も is sometimes called a "case-marking particle" rather than a "sentence particle."
- **Scalar も with time can be nostalgic or burdensome.** <ruby>三<rt>さん</rt></ruby><ruby>時間<rt>じかん</rt></ruby>も<ruby>待<rt>ま</rt></ruby>った sounds long and burdensome. <ruby>三<rt>さん</rt></ruby><ruby>時間<rt>じかん</rt></ruby>も<ruby>話<rt>はな</rt></ruby>した sounds long and possibly intimate (we talked for a whole three hours). The emotional valence depends on the verb.
- **<ruby>何<rt>なん</rt></ruby>でも and <ruby>誰<rt>だれ</rt></ruby>でも are affirmative universals.** Unlike <ruby>何<rt>なん</rt></ruby>も/<ruby>誰<rt>だれ</rt></ruby>も which require negatives, <ruby>何<rt>なん</rt></ruby>でも<ruby>食<rt>た</rt></ruby>べる means "I eat anything" and <ruby>誰<rt>だれ</rt></ruby>でも<ruby>来<rt>き</rt></ruby>ていい means "Anyone may come." The でも here is the conditional/extreme particle, not the conjunction.
- **いつも is the lexicalized exception.** It behaves like an adverb meaning "always" and does not require negative polarity. It is essentially a different word that happens to share the spelling.
- **〜てもいい is a permission pattern with hidden complexity.** <ruby>行<rt>い</rt></ruby>ってもいい ("You may go") uses も to mark the action as extreme — "Even if you go, it's fine." Over time this has grammaticalized into the standard permission pattern. But the scalar origin is still audible: the speaker is saying "Going is at the edge of acceptable actions, and I'm okay with it."
- **〜もしない with positive verbs means "not even."** <ruby>笑<rt>わら</rt></ruby>いもしない ("doesn't even laugh") is stronger than <ruby>笑<rt>わら</rt></ruby>わない ("doesn't laugh"). The も exhausts the set of possible reactions.
- **The "both… and…" pattern (AもBも) implies exhaustiveness.** When you say <ruby>猫<rt>ねこ</rt></ruby>も<ruby>犬<rt>いぬ</rt></ruby>も<ruby>好<rt>す</rt></ruby>き, you imply these are the relevant categories. Adding <ruby>鳥<rt>とり</rt></ruby>も changes the implied set. The listener unconsciously tracks what も marks as "the complete relevant group."

## Learner Stage {#learner-stage}

| Stage | What It Looks Like |
|-------|-------------------|
| **Now — N5** | Use additive も correctly (<ruby>私<rt>わたくし</rt></ruby>も, AもBも); avoid double-marking with は/が; recognize いつも as "always" |
| **Developing — N5→N4** | Deploy scalar も with quantities (<ruby>三<rt>さん</rt></ruby><ruby>時間<rt>じかん</rt></ruby>も, <ruby>千<rt>せん</rt></ruby><ruby>円<rt>えん</rt></ruby>も); master <ruby>何<rt>なん</rt></ruby>も/<ruby>誰<rt>だれ</rt></ruby>も/どこも + negative; use 〜ても/〜でも for conditional extremes |
| **Automatic — N4→N3** | Distinguish additive も from scalar も from extreme も in real time; use <ruby>子供<rt>こども</rt></ruby>でも naturally; understand 〜てもいい as permission; produce <ruby>一<rt>いち</rt></ruby><ruby>度<rt>ど</rt></ruby>も〜ない without thinking |
| **Mastery** | You control the full も spectrum — adding items, scaling quantities, exhausting universes, and pushing conditional boundaries. You know when さえ would be too sharp, when だけ would be too exclusive, and when も is exactly right. You also hear the scalar origin in てもいい permission and the lexicalized drift in いつも. |

## See Also {#see-also}

### Same Family

- は (topic marker) — the particle も often replaces
- が (subject marker) — the particle も often replaces
- を (object marker) — the particle も often replaces
- に / で / から — particles that stack with も

### Commonly Confused With

- でも (sentence-initial) — "but" — unrelated conjunction
- さえ / すら — "even" — more extreme and often sharper than も
- だけ — "only" — exclusive opposite of additive も
- しか〜ない — "only" — negative-frame exclusive
- とても — "very" — adverb, unrelated to conditional ても

### Builds From

- は・が・を (N5) — the core particles も replaces
- て-form (N4) — required for conditional も
- Wh-words (N5) — <ruby>何<rt>なん</rt></ruby>, <ruby>誰<rt>だれ</rt></ruby>, どこ, etc.
- Negative verbs (N5) — required for universal negative も

### Unlocks Next

- さえ / すら (N3) — sharper extremes
- 〜も〜ば〜も (N2) — correlative "both… and… / neither… nor…"
- 〜ても<ruby>仕方<rt>しかた</rt></ruby>がない (N3) — "even if… it can't be helped"
- 〜なくてもいい (N4) — negative permission; "don't have to"
- <ruby>何<rt>なん</rt></ruby>でも (N4) — affirmative universal; "anything / everything"
