# Respectful Request — お/ご〜ください

## ご<ruby>覧<rt>らん</rt></ruby>ください。 / お<ruby>座<rt>すわ</rt></ruby>りください。

---

## JLPT Level {#jlpt-level}

!!! abstract "JLPT Level — Upper N4 / Lower N3"
    The pattern itself is N4 — a core <ruby>尊敬語<rt>そんけいご</rt></ruby> (sonkeigo / respectful language) structure. But the calibration of when to use it versus neutral 〜てください or directive 〜なさい, and the wago/kango prefix distinction, push it into lower-N3 territory for active production. Service-industry fluency depends on this pattern.

    | Tier | Pattern | Description | Example |
    |------|---------|-------------|---------|
    | N5 — Foundation | 〜てください | Neutral request — "please do X" | <ruby>座<rt>すわ</rt></ruby>ってください。 |
    | N4 — Foundation | 〜なさい | Directive — "do X" (authority to subordinate) | <ruby>座<rt>すわ</rt></ruby>りなさい。 |
    | **N4 — Core ← You are here** | **お/ご〜ください** | **Respectful request — elevates the listener's action** | お<ruby>座<rt>すわ</rt></ruby>りください。 |
    | N4 — Foundation | お/ご〜する | Humble — lowers speaker's action | お<ruby>座<rt>すわ</rt></ruby>りします。 |
    | N3 — Extended | お/ご〜いただけますか | Humble request for listener's action | お<ruby>座<rt>すわ</rt></ruby>りいただけますか。 |
    | N2 — Awareness | お/ご〜になられる | Double keigo (often avoided) | お<ruby>座<rt>すわ</rt></ruby>りになられる。 |

## Quick Summary {#quick-summary}

!!! tip "At a glance"
    - お/ご〜ください is a **<ruby>尊敬語<rt>そんけいご</rt></ruby> (sonkeigo)** pattern: it elevates the listener's action while politely requesting it. It does not mean "please" in a casual sense — it means "I humbly ask that you, who are above me, do this elevated action."
    - **お** attaches to **<ruby>和語<rt>わご</rt></ruby> (wago / native Japanese)** verb stems: <ruby>座<rt>すわ</rt></ruby>る → お<ruby>座<rt>すわ</rt></ruby>りください, <ruby>休<rt>やす</rt></ruby>む → お<ruby>休<rt>やす</rt></ruby>みください.
    - **ご** attaches to **<ruby>漢語<rt>かんご</rt></ruby> (kango / Sino-Japanese)** verb stems or nouns: <ruby>利用<rt>りよう</rt></ruby>する → ご<ruby>利用<rt>りよう</rt></ruby>ください, <ruby>覧<rt>らん</rt></ruby>する → ご<ruby>覧<rt>らん</rt></ruby>ください.
    - This pattern is **customer-facing, service-industry, and upward-directed**. You use it when speaking to guests, clients, superiors, or anyone whose status you want to honor.
    - It is **not** the humble form. お<ruby>座<rt>すわ</rt></ruby>りします (I will sit — humble) and お<ruby>座<rt>すわ</rt></ruby>りください (Please sit — respectful) are opposite social directions. Confusing them is a major keigo error.

## Core Meaning {#core-meaning}

!!! note "Core Rule"
    Use お/ご〜ください when you want to request that someone of higher status or someone you are serving perform an action, while grammatically elevating that action. The request is polite but structurally framed as an honor being granted to you by their action.

| お<ruby>座<rt>すわ</rt></ruby>りください。 | Please be seated [respectful, to a guest or superior]. |
|---|---|
| | *Literal: Humble-prefix + sit-stem + please-do [respectful].* |
| | *Context: Restaurant host to customer. Doctor to patient. Receptionist to visitor. The host elevates the act of sitting.* |

| ご<ruby>覧<rt>らん</rt></ruby>ください。 | Please take a look [respectful, to a client or superior]. |
|---|---|
| | *Literal: Humble-prefix + see-stem + please-do [respectful].* |
| | *Context: Shop staff showing merchandise. Presenter indicating a slide. The act of looking is elevated.* |

| ご<ruby>確認<rt>かくにん</rt></ruby>ください。 | Please confirm [respectful, business correspondence]. |
|---|---|
| | *Literal: Humble-prefix + confirm-stem + please-do [respectful].* |
| | *Context: Email to a client. The writer elevates the client's act of checking.* |

| お<ruby>召<rt>め</rt></ruby>し<ruby>上<rt>あ</rt></ruby>がりください。 | Please partake / Please eat [ultra-respectful, service]. |
|---|---|
| | *Literal: Humble-prefix + eat-stem + please-do [respectful].* |
| | *Context: High-end restaurant. <ruby>召<rt>め</rt></ruby>し<ruby>上<rt>あ</rt></ruby>がる is the respectful verb for "eat"; adding お/ご〜ください double-elevates.* |

## In Real Life {#in-real-life}

!!! example "In Real Life"
    **Situation:** ユミ works at the front desk of a hotel. A guest has just completed check-in. She gestures toward the elevator bank and speaks.

    &gt; エレベーターはあちらでございます。どうぞ、お<ruby>掛<rt>か</rt></ruby>けになりまして、お<ruby>休<rt>やす</rt></ruby>みください。

    *"The elevator is over there. Please make yourself comfortable and rest."*

    **Why お<ruby>休<rt>やす</rt></ruby>みください fits here:** Yumi could have said <ruby>休<rt>やす</rt></ruby>んでください — grammatically correct, neutrally polite. But that would treat the guest as a peer. お<ruby>休<rt>やす</rt></ruby>みください elevates the guest's rest into an honored action. The お marks the verb as wago; the ください frames it as a respectful request.

    **What makes this natural:** The pattern appears in the standard hospitality script. It is not a spontaneous sentence — it is a social ritual encoded in grammar. The guest expects this level of elevation; anything less feels like a downgrade.

    **The どうぞ that precedes:** どうぞ ("by all means") is a common partner for お/ご〜ください. It adds warm invitation to the respectful structure. どうぞお<ruby>座<rt>すわ</rt></ruby>りください is a standard pairing.

## How It Works {#how-it-works}

The pattern is built from three layers: prefix + stem + request marker. Each layer carries social meaning.

**Layer 1 — Prefix (お vs ご):** The prefix elevates the following word. お is for native Japanese vocabulary; ご is for Sino-Japanese. This is not optional — using the wrong prefix is immediately recognizable as incorrect.

**Layer 2 — Stem:** The verb is reduced to its stem form (the form used before ます). For する verbs, the noun/stem is used (<ruby>利用<rt>りよう</rt></ruby>する → <ruby>利用<rt>りよう</rt></ruby>). This stripped form is then "wrapped" in the honorific prefix.

**Layer 3 — ください:** This is the imperative form of くださる, the respectful verb "to give (to me/someone below)." By using ください, the speaker frames the action as something the honored listener will "give" by performing it. The social geometry is: "You will do this for me, and I am grateful."

| <ruby>座<rt>すわ</rt></ruby>ってください。 | Please sit [neutral request]. |
|---|---|
| | *The listener is a peer or the situation is casual. No elevation.* |

| お<ruby>座<rt>すわ</rt></ruby>りください。 | Please be seated [respectful request]. |
|---|---|
| | *The listener is a guest or superior. Their sitting is elevated.* |

| お<ruby>座<rt>すわ</rt></ruby>りします。 | I will be seated [humble]. |
|---|---|
| | *The speaker lowers their own action. Social direction is reversed.* |

### Real Speech vs. Textbook Form

Textbooks often present お/ご〜ください as "please do X." In real speech, it is a **service-industry default** and an **upward-deference marker**. It is less common in peer conversation than in structured service contexts.

| Form | Example | Note |
|------|---------|------|
| Textbook (neutral) | <ruby>窓<rt>まど</rt></ruby>を<ruby>開<rt>あ</rt></ruby>けてください。 | "Please open the window" — classroom example |
| Real speech (respectful) | <ruby>窓<rt>まど</rt></ruby>をお<ruby>開<rt>あ</rt></ruby>けください。 | "Please open the window" — hotel staff to guest |
| Real speech (humble request) | <ruby>窓<rt>まど</rt></ruby>をお<ruby>開<rt>あ</rt></ruby>けいただけますか。 | "Could you open the window for me?" — even more deferential |
| Real speech (directive) | <ruby>窓<rt>まど</rt></ruby>を<ruby>開<rt>あ</rt></ruby>けなさい。 | "Open the window" — parent to child |

## Grammar Analysis {#grammar-analysis}

### Function in Context

- お<ruby>座<rt>すわ</rt></ruby>りください: <ruby>座<rt>すわ</rt></ruby>る (to sit) is wago, so it takes お. The stem is <ruby>座<rt>すわ</rt></ruby>り. ください elevates the imperative.
- ご<ruby>覧<rt>らん</rt></ruby>ください: <ruby>覧<rt>らん</rt></ruby>する (to view) is kango, so it takes ご. The stem/noun is <ruby>覧<rt>らん</rt></ruby>. ください elevates the imperative.
- ご<ruby>利用<rt>りよう</rt></ruby>ください: <ruby>利用<rt>りよう</rt></ruby>する (to use) is kango, so it takes ご. The stem/noun is <ruby>利用<rt>りよう</rt></ruby>.
- お<ruby>召<rt>め</rt></ruby>し<ruby>上<rt>あ</rt></ruby>がりください: <ruby>召<rt>め</rt></ruby>し<ruby>上<rt>あ</rt></ruby>がる (to eat — respectful verb) is wago, so it takes お. The stem is <ruby>召<rt>め</rt></ruby>し<ruby>上<rt>あ</rt></ruby>がり. This is already a respectful verb; adding お/ご〜ください makes it ultra-respectful.
- The subject of these sentences is always the listener (you), but it is never stated. The grammar of <ruby>尊敬語<rt>そんけいご</rt></ruby> implies "you, the honored person."
- Without the prefix: <ruby>座<rt>すわ</rt></ruby>りください is ungrammatical. The お/ご is required for the respectful frame.

| お<ruby>掛<rt>か</rt></ruby>けください。 | Please sit down [hanging / seated position]. |
|---|---|
| | *Context: <ruby>掛<rt>か</rt></ruby>ける (to hang / to set) in お<ruby>掛<rt>か</rt></ruby>けになる means "to seat oneself." お<ruby>掛<rt>か</rt></ruby>けください is a slightly more formal variant of お<ruby>座<rt>すわ</rt></ruby>りください in some service contexts.* |

!!! warning "The wago/kango boundary"
    Most errors involve applying お to kango or ご to wago. If the word can be written in pure hiragana or has native Japanese etymology, it likely takes お. If it is a two-kanji compound or clearly Sino-Japanese, it takes ご.

!!! warning "Some words are lexicalized chunks"
    ご<ruby>覧<rt>らん</rt></ruby>, ご<ruby>注意<rt>ちゅうい</rt></ruby>, お<ruby>待<rt>ま</rt></ruby>ち are so common they should be learned as complete units rather than assembled from rules each time.

!!! warning "Do not use お/ご〜ください for your own actions"
    This is <ruby>尊敬語<rt>そんけいご</rt></ruby> (respectful), not <ruby>謙譲語<rt>けんじょうご</rt></ruby> (humble). It points upward at the listener. Using it for yourself (お<ruby>座<rt>すわ</rt></ruby>りください when you are the one sitting) is a social-direction error.

## Japanese Mindset {#japanese-mindset}

Japanese keigo is not "polite Japanese" — it is a **social positioning system**. お/ご〜ください does more than soften a command. It restructures the relationship between speaker and listener.

When a hotel clerk says お<ruby>座<rt>すわ</rt></ruby>りください, they are not merely saying "Sit down." They are saying: "You are a guest of high standing, and your act of sitting is an honor. I am grateful that you will do it." The grammar encodes an entire service philosophy.

- **Elevation as hospitality.** The prefix お/ご lifts the listener's action out of the everyday. Sitting becomes "being seated." Looking becomes "viewing." Eating becomes "partaking." The everyday is ceremonialized.
- **Request as gift.** ください derives from くださる (to give from above). The listener is framed as giving the action to the speaker. Even when the listener is clearly following an instruction, the grammar treats them as bestowing a favor.
- **The wago/kango split as cultural taxonomy.** The fact that native and Sino-Japanese vocabulary take different prefixes reveals how deeply Japanese grammar encodes the origin of words. The particle system respects etymology.

## Nuance Comparison {#nuance-comparison}

| Grammar Point | Core Meaning | Social Direction | Register | Trigger Context | JLPT |
|---|---|---|---|---|---|
| お/ご〜ください | "Please do X" [respectful] | Elevates listener | Formal, service, upward | Speaking to guests, clients, superiors | N4 |
| 〜てください | "Please do X" [neutral] | Neutral request | Polite, general | Peers, general requests | N5 |
| 〜なさい | "Do X" [directive] | Speaker directs listener | Authority to subordinate | Parent to child, teacher to student | N4 |
| お/ご〜する | "I do X" [humble] | Lowers speaker | Humble | Describing one's own actions to superiors | N4 |
| お/ご〜いただけますか | "Could you do X?" [humble request] | Humble request for listener's action | Ultra-polite | Asking superiors for permission/action | N3 |
| お/ご〜なさい | "Do X" [respectful directive] | Elevated directive | Formal, slightly old-fashioned | Formal instruction, written | N4 |
| 〜ていただけますか | "Could I receive X?" [humble] | Receiving frame | Polite | Requesting someone do something for you | N3 |

**お/ご〜ください vs 〜てください:** Both mean "please do." The difference is elevation. <ruby>座<rt>すわ</rt></ruby>ってください is a polite request to a peer. お<ruby>座<rt>すわ</rt></ruby>りください is a respectful request to a guest. Using 〜てください with a hotel guest sounds slightly brusque; using お/ご〜ください with a close friend sounds oddly ceremonial.

**お/ご〜ください vs お/ご〜する:** These are opposite directions. お<ruby>座<rt>すわ</rt></ruby>りください = "You, please sit" (respectful). お<ruby>座<rt>すわ</rt></ruby>りします = "I will sit" (humble). The prefix is the same; the social direction is reversed by the main verb (ください vs する).

## Alternatives {#alternatives}

### 〜てください (neutral request)

The standard polite request. No elevation.

| | |
|---|---|
| **Example** | <ruby>座<rt>すわ</rt></ruby>ってください。 |
| **English** | "Please sit." |
| **Intent** | Polite but neutral. The listener is a peer or the context is casual. |
| **vs. お/ご〜ください** | Removes the ceremonial elevation. Appropriate for colleagues, friends, or everyday service. |

!!! tip "When to use this instead"
    Use 〜てください when speaking to peers, subordinates in casual contexts, or when the situation does not require service-industry formality.

### 〜なさい (directive)

Authority speaking downward. Not polite in the deferential sense — it is a command wrapped in grammar.

| | |
|---|---|
| **Example** | <ruby>座<rt>すわ</rt></ruby>りなさい。 |
| **English** | "Sit down." |
| **Intent** | Directive. The speaker has authority. |
| **vs. お/ご〜ください** | なさい commands; お/ご〜ください requests. Never use なさい with guests or superiors. |

### お/ご〜いただけますか (humble request)

Asks the listener to "give" the action in an even more humble frame.

| | |
|---|---|
| **Example** | お<ruby>座<rt>すわ</rt></ruby>りいただけますか。 |
| **English** | "Could you please be seated?" |
| **Intent** | The speaker positions themselves as receiving the listener's action. More humble than お<ruby>座<rt>すわ</rt></ruby>りください. |
| **vs. お/ご〜ください** | いただけますか adds a receiving frame (<ruby>謙譲語<rt>けんじょうご</rt></ruby> layer on top of <ruby>尊敬語<rt>そんけいご</rt></ruby>). Appropriate for very high-status listeners or formal written requests. |

### お/ご〜なさい (respectful directive)

A middle ground between request and directive. Slightly old-fashioned.

| | |
|---|---|
| **Example** | お<ruby>休<rt>やす</rt></ruby>みなさい。 |
| **English** | "Do rest." / "Good night." |
| **Intent** | Elevated directive. Common in set phrases (お<ruby>休<rt>やす</rt></ruby>みなさい = good night). Less common for spontaneous requests. |
| **vs. お/ご〜ください** | なさい is more decisive; ください is more inviting. |

## Register and Formality {#register-and-formality}

The respectful request is formal by definition, but its warmth can be modulated.

=== ":material-chat-outline: Casual (inappropriate)"
    お<ruby>座<rt>すわ</rt></ruby>りください。

    *"Please be seated."*

    **Shift:** Used with a close friend. Sounds mock-formal or sarcastic unless the context is genuinely ceremonial.

=== ":material-tie: Basic Polite (service standard)"
    どうぞ、お<ruby>座<rt>すわ</rt></ruby>りください。

    *"Please, be seated."*

    **Shift:** Standard service-industry warmth. The どうぞ adds invitation; the お<ruby>座<rt>すわ</rt></ruby>りください provides the elevation.

=== ":material-briefcase-outline: Extra Polite (business)"
    ご<ruby>確認<rt>かくにん</rt></ruby>のほど、よろしくお<ruby>願<rt>ねが</rt></ruby>いいたします。

    *"We kindly ask that you please confirm."*

    **Shift:** ご<ruby>確認<rt>かくにん</rt></ruby> + のほど (formal periphrasis) + よろしくお<ruby>願<rt>ねが</rt></ruby>いいたします (humble closing). The core pattern ご<ruby>確認<rt>かくにん</rt></ruby>ください is embedded in a more elaborate formal structure.

=== ":material-crown: Ultra-Formal (high-end hospitality)"
    どうぞ、お<ruby>召<rt>め</rt></ruby>し<ruby>上<rt>あ</rt></ruby>がりくださいませ。

    *"Please, do partake."*

    **Shift:** ください + ませ adds historical/formal flourish. Found in luxury ryokan, traditional restaurants, and theatrical speech.

## Common Mistakes {#common-mistakes}

### 1. Using お with kango or ご with wago

!!! failure "Incorrect prefix"
    お<ruby>利用<rt>りよう</rt></ruby>ください。

    ~<ruby>利用<rt>りよう</rt></ruby> is kango (Sino-Japanese). It requires ご, not お.~

!!! success "Correct"
    ご<ruby>利用<rt>りよう</rt></ruby>ください。

!!! failure "Incorrect prefix"
    ご<ruby>座<rt>すわ</rt></ruby>りください。

    ~<ruby>座<rt>すわ</rt></ruby>る is wago (native Japanese). It requires お, not ご.~

!!! success "Correct"
    お<ruby>座<rt>すわ</rt></ruby>りください。

### 2. Confusing <ruby>尊敬語<rt>そんけいご</rt></ruby> (respectful) with <ruby>謙譲語<rt>けんじょうご</rt></ruby> (humble)

!!! failure "Social direction error"
    お<ruby>座<rt>すわ</rt></ruby>りします → said to a guest as an invitation to sit.

    ~This means "I will sit" (humble). You have just announced that you, the staff member, will be sitting. The guest is left standing.~

!!! success "Correct"
    お<ruby>座<rt>すわ</rt></ruby>りください。 = "Please sit" (respectful, to guest).
    お<ruby>座<rt>すわ</rt></ruby>りします。 = "I will sit" (humble, about self).

### 3. Using お/ご〜ください in casual peer contexts

!!! failure "Register mismatch"
    <ruby>友達<rt>ともだち</rt></ruby>に「お<ruby>座<rt>すわ</rt></ruby>りください」

    *"Please be seated, friend."* — Sounds mock-formal or sarcastic. Friends say <ruby>座<rt>すわ</rt></ruby>って or <ruby>座<rt>すわ</rt></ruby>ってよ.

!!! success "Appropriate"
    <ruby>座<rt>すわ</rt></ruby>って。 / <ruby>座<rt>すわ</rt></ruby>ってください。 (neutral/casual polite)

### 4. Using polite ます stem instead of plain stem

!!! failure "Stem error"
    お<ruby>座<rt>すわ</rt></ruby>りますください。

    ~Ungrammatical. The stem is <ruby>座<rt>すわ</rt></ruby>り, not <ruby>座<rt>すわ</rt></ruby>ります.~

!!! success "Correct"
    お<ruby>座<rt>すわ</rt></ruby>りください。

## Hidden Points {#hidden-points}

- **お/ご〜ください can be used in written public notices without a specific addressee.** ご<ruby>注意<rt>ちゅうい</rt></ruby>ください on a sign addresses the general public. The respectful frame treats any reader as an honored guest.
- **The pattern is often shortened in very casual respectful speech.** A host might say どうぞ、お<ruby>座<rt>すわ</rt></ruby>り — dropping ください but retaining the elevation. This is warm and intimate rather than incomplete.
- **Some verbs resist the pattern.** Verbs that are inherently humble or already respectful may sound redundant or odd with お/ご〜ください. いらっしゃる (respectful "to go/come") does not take お/ご〜ください; you say いらっしゃってください or simply お<ruby>越<rt>こ</rt></ruby>しください.
- **Double keigo (お/ご〜なさいます / お/ご〜なられる) exists but is often avoided.** お<ruby>座<rt>すわ</rt></ruby>りになられます is grammatically possible but sounds excessive to many native speakers. It is used in ultra-formal contexts but can feel like overcorrection.
- **お/ご〜くださいませ is a historical/formal variant.** Adding ませ (imperative of ます) creates an even more elevated tone: お<ruby>座<rt>すわ</rt></ruby>りくださいませ. This survives in luxury hospitality and period drama.
- **The pattern can be used sarcastically or aggressively.** With the wrong intonation, お<ruby>帰<rt>かえ</rt></ruby>りください can mean "Please leave" (dismissive). The respectful grammar wrapped around a rejection creates social dissonance — a classic Japanese passive-aggressive move.
- **お<ruby>待<rt>ま</rt></ruby>ちください is often used when the speaker is actually making the listener wait.** The grammar treats the wait as an honored action the listener is performing, which softens the imposition. This is keigo as emotional management.
