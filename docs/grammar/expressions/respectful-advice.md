# <ruby>敬<rt>けい</ruby>意<rt>い</rt>ある<rt>あ</rt><ruby>依<rt>い</rt></ruby>頼 — お/ご〜<ruby>下<rt>くだ</rt></ruby>さい

## ご<ruby>覧<rt>らん</rt></ruby><ruby>下<rt>くだ</rt></ruby>さい。 / お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい。

---

## JLPT <ruby>Level<rt>レベル</rt></ruby> {#jlpt-level}

!!! abstract "JLPT <ruby>Level<rt>レベル</rt></ruby> — Upper N4 / Lower N3"
    The pattern itself is N4 — a core <ruby>尊敬語<rt>そんけいご</rt></ruby> (<ruby>sonkeigo<rt>そんけいご</rt></ruby> / respectful language) structure. But the calibration of when to use it versus neutral 〜て<ruby>下<rt>くだ</rt></ruby>さい or directive 〜<ruby>なさい<rt>なさい</rt></ruby>, and the wago/kango prefix distinction, push it into lower-N3 territory for active production. Service-industry fluency depends on this pattern.

    | Tier | Pattern | Description | Example |
    |------|---------|-------------|---------|
    | N5 — Foundation | 〜て<ruby>下<rt>くだ</rt></ruby>さい | Neutral request — "please do X" | <ruby>座<rt>すわ</rt></ruby>って<ruby>下<rt>くだ</rt></ruby>さい。 |
    | N4 — Foundation | 〜<ruby>なさい<rt>なさい</rt></ruby> | Directive — "do X" (authority to subordinate) | <ruby>座<rt>すわ</rt></ruby>り<ruby>なさい<rt>なさい</rt></ruby>。 |
    | **N4 — Core ← You are here** | **お/ご〜<ruby>下<rt>くだ</rt></ruby>さい** | **Respectful request — elevates the listener's action** | お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい。 |
    | N4 — Foundation | お/ご〜<ruby>する<rt>する</rt></ruby> | Humble — lowers speaker's action | お<ruby>座<rt>すわ</rt></ruby>り<ruby>します<rt>します</rt></ruby>。 |
    | N3 — Extended | お/ご〜いただけますか | Humble request for listener's action | お<ruby>座<rt>すわ</rt></ruby>りいただけますか。 |
    | N2 — Awareness | お/ご〜になられる | Double keigo (often avoided) | お<ruby>座<rt>すわ</rt></ruby>りになられる。 |

## <ruby>Quick<rt>クイック</rt></ruby> <ruby>Summary<rt>サマリー</rt></ruby> {#quick-summary}

!!! tip "At a glance"
    - お/ご〜<ruby>下<rt>くだ</rt></ruby>さい is a **<ruby>尊敬語<rt>そんけいご</rt></ruby> (<ruby>sonkeigo<rt>そんけいご</rt></ruby>)** pattern: it elevates the listener's action while politely requesting it. It does not mean "please" in a casual sense — it means "I humbly ask that you, who are above me, do this elevated action."
    - **お** attaches to **<ruby>和語<rt>わご</rt></ruby> (<ruby>wago<rt>わご</rt></ruby> / native <ruby>Japanese<rt>じゃぱにーず</rt></ruby>)** verb stems: <ruby>座<rt>すわ</rt></ruby>る → お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい, <ruby>休<rt>やす</rt></ruby>む → お<ruby>休<rt>やす</rt></ruby>み<ruby>下<rt>くだ</rt></ruby>さい.
    - **ご** attaches to **<ruby>漢語<rt>かんご</rt></ruby> (<ruby>kango<rt>かんご</rt></ruby> / <ruby>Sino<rt>しの</rt></ruby>-<ruby>Japanese<rt>じゃぱにーず</rt></ruby>)** verb stems or nouns: <ruby>利用<rt>りよう</rt></ruby>する → ご<ruby>利用<rt>りよう</rt></ruby><ruby>下<rt>くだ</rt></ruby>さい, <ruby>覧<rt>らん</rt></ruby>する → ご<ruby>覧<rt>らん</rt></ruby><ruby>下<rt>くだ</rt></ruby>さい.
    - This pattern is **customer-facing, service-industry, and upward-directed**. You use it when speaking to guests, clients, superiors, or anyone whose status you want to honor.
    - It is **not** the humble form. お<ruby>座<rt>すわ</rt></ruby>り<ruby>します<rt>します</rt></ruby> (I will sit — humble) and お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい (Please sit — respectful) are opposite social directions. Confusing them is a major keigo error.

## <ruby>Core<rt>こあ</rt></ruby> <ruby>Meaning<rt>いみ</rt></ruby> {#core-meaning}

!!! note "Core Rule"
    Use お/ご〜<ruby>下<rt>くだ</rt></ruby>さい when you want to request that someone of higher status or someone you are serving perform an action, while grammatically elevating that action. The request is polite but structurally framed as an honor being granted to you by their action.

| お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい。 | Please be seated [respectful, to a guest or superior]. |
|---|---|
| | *<ruby>Literal<rt>りたてらる</rt></ruby>: <ruby>Humble<rt>へんきん</rt></ruby>-prefix + sit-stem + please-do [respectful].* |
| | *<ruby>Context<rt>こんてくすと</rt></ruby>: <ruby>Restaurant<rt>れすとらん</rt></ruby> host to customer. <ruby>Doctor<rt>どくたー</rt></ruby> to patient. <ruby>Receptionist<rt>れせぷしょにすと</rt></ruby> to visitor. The host elevates the act of sitting.* |

| ご<ruby>覧<rt>らん</rt></ruby><ruby>下<rt>くだ</rt></ruby>さい。 | Please take a look [respectful, to a client or superior]. |
|---|---|
| | *<ruby>Literal<rt>りたてらる</rt></ruby>: <ruby>Humble<rt>へんきん</rt></ruby>-prefix + see-stem + please-do [respectful].* |
| | *<ruby>Context<rt>こんてくすと</rt></ruby>: <ruby>Shop<rt>しょっぷ</rt></ruby> staff showing merchandise. <ruby>Presenter<rt>ぷれぜんたー</rt></ruby> indicating a slide. The act of looking is elevated.* |

| ご<ruby>確認<rt>かくにん</rt></ruby><ruby>下<rt>くだ</rt></ruby>さい。 | Please confirm [respectful, business correspondence]. |
|---|---|
| | *<ruby>Literal<rt>りたてらる</rt></ruby>: <ruby>Humble<rt>へんきん</rt></ruby>-prefix + confirm-stem + please-do [respectful].* |
| | *<ruby>Context<rt>こんてくすと</rt></ruby>: <ruby>Email<rt>いーめーる</rt></ruby> to a client. The writer elevates the client's act of checking.* |

| お<ruby>召<rt>め</rt></ruby>し<ruby>上<rt>あ</rt></ruby>がり<ruby>下<rt>くだ</rt></ruby>さい。 | Please partake / Please eat [ultra-respectful, service]. |
|---|---|
| | *<ruby>Literal<rt>りたてらる</rt></ruby>: <ruby>Humble<rt>へんきん</rt></ruby>-prefix + eat-stem + please-do [respectful].* |
| | *<ruby>Context<rt>こんてくすと</rt></ruby>: <ruby>High<rt>はい</rt></ruby>-end <ruby>restaurant<rt>れすとらん</rt></ruby>. <ruby>召<rt>め</rt></ruby>し<ruby>上<rt>あ</rt></ruby>がる is the respectful verb for "eat"; adding お/ご〜<ruby>下<rt>くだ</rt></ruby>さい double-elevates.* |

## In <ruby>Real<rt>りある</rt></ruby> <ruby>Life<rt>らいふ</rt></ruby> {#in-real-life}

!!! example "In Real Life"
    **<ruby>Situation<rt>しちゅえーしょん</rt></ruby>:** ユミ works at the front desk of a hotel. A guest has just completed check-in. She gestures toward the elevator bank and speaks.

    &gt; エレベーターはあちらでございます。どうぞ、お<ruby>掛<rt>か</rt></ruby>けになりまして、お<ruby>休<rt>やす</rt></ruby>み<ruby>下<rt>くだ</rt></ruby>さい。

    *"The elevator is over there. Please make yourself comfortable and rest."*

    **Why お<ruby>休<rt>やす</rt></ruby>み<ruby>下<rt>くだ</rt></ruby>さい fits here:** Yumi could have said <ruby>休<rt>やす</rt></ruby>んで<ruby>下<rt>くだ</rt></ruby>さい — grammatically correct, neutrally polite. But that would treat the guest as a peer. お<ruby>休<rt>やす</rt></ruby>み<ruby>下<rt>くだ</rt></ruby>さい elevates the guest's rest into an honored action. The お marks the verb as wago; the <ruby>下<rt>くだ</rt></ruby>さい frames it as a respectful request.

    **What makes this natural:** The pattern appears in the standard hospitality script. It is not a spontaneous sentence — it is a social ritual encoded in grammar. The guest expects this level of elevation; anything less feels like a downgrade.

    **The どうぞ that precedes:** どうぞ ("by all means") is a common partner for お/ご〜<ruby>下<rt>くだ</rt></ruby>さい. It adds warm invitation to the respectful structure. どうぞお<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい is a standard pairing.

## How It <ruby>Works<rt>わーくす</rt></ruby> {#how-it-works}

The pattern is built from three layers: prefix + stem + request marker. Each layer carries social meaning.

**<ruby>Layer<rt>れいやー</rt></ruby> 1 — Prefix (お vs ご):** The prefix elevates the following word. お is for native <ruby>Japanese<rt>じゃぱにーず</rt></ruby> vocabulary; ご is for <ruby>Sino<rt>しの</rt></ruby>-<ruby>Japanese<rt>じゃぱにーず</rt></ruby>. This is not optional — using the wrong prefix is immediately recognizable as incorrect.

**<ruby>Layer<rt>れいやー</rt></ruby> 2 — Stem:** The verb is reduced to its stem form (the form used before ます). For する verbs, the noun/stem is used (<ruby>利用<rt>りよう</rt></ruby>する → <ruby>利用<rt>りよう</rt></ruby>). This stripped form is then "wrapped" in the honorific prefix.

**<ruby>Layer<rt>れいやー</rt></ruby> 3 — <ruby>下<rt>くだ</rt></ruby>さい:** This is the imperative form of <ruby>下<rt>くだ</rt></ruby>さる, the respectful verb "to give (to me/someone below)." By using <ruby>下<rt>くだ</rt></ruby>さい, the speaker frames the action as something the honored listener will "give" by performing it. The social geometry is: "You will do this for me, and I am grateful."

| <ruby>座<rt>すわ</rt></ruby>って<ruby>下<rt>くだ</rt></ruby>さい。 | Please sit [neutral request]. |
|---|---|
| | *The listener is a peer or the situation is casual. No elevation.* |

| お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい。 | Please be seated [respectful request]. |
|---|---|
| | *The listener is a guest or superior. Their sitting is elevated.* |

| お<ruby>座<rt>すわ</rt></ruby>り<ruby>します<rt>します</rt></ruby>。 | I will be seated [humble]. |
|---|---|
| | *The speaker lowers their own action. Social direction is reversed.* |

### <ruby>Real<rt>りある</rt></ruby> <ruby>Speech<rt>すぴーち</rt></ruby> vs. <ruby>Textbook<rt>てくすとぶっく</rt></ruby> <ruby>Form<rt>ふぉーむ</rt></ruby>

<ruby>Textbooks<rt>てくすとぶっくす</rt></ruby> often present お/ご〜<ruby>下<rt>くだ</rt></ruby>さい as "please do X." In real speech, it is a **service-industry default** and an **upward-deference marker**. It is less common in peer conversation than in structured service contexts.

| Form | Example | Note |
|------|---------|------|
| <ruby>Textbook<rt>てくすとぶっく</rt></ruby> (neutral) | <ruby>窓<rt>まど</rt></ruby>を<ruby>開<rt>あ</rt></ruby>けて<ruby>下<rt>くだ</rt></ruby>さい。 | "Please open the window" — classroom example |
| <ruby>Real<rt>りある</rt></ruby> speech (respectful) | <ruby>窓<rt>まど</rt></ruby>をお<ruby>開<rt>あ</rt></ruby>け<ruby>下<rt>くだ</rt></ruby>さい。 | "Please open the window" — hotel staff to guest |
| <ruby>Real<rt>りある</rt></ruby> speech (humble request) | <ruby>窓<rt>まど</rt></ruby>をお<ruby>開<rt>あ</rt></ruby>けいただけますか。 | "Could you open the window for me?" — even more deferential |
| <ruby>Real<rt>りある</rt></ruby> speech (directive) | <ruby>窓<rt>まど</rt></ruby>を<ruby>開<rt>あ</rt></ruby>け<ruby>なさい<rt>なさい</rt></ruby>。 | "Open the window" — parent to child |

## <ruby>Grammar<rt>ぐらまー</rt></ruby> <ruby>Analysis<rt>あなりしす</rt></ruby> {#grammar-analysis}

### <ruby>Function<rt>ふんくしょん</rt></ruby> in <ruby>Context<rt>こんてくすと</rt></ruby>

- お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい: <ruby>座<rt>すわ</rt></ruby>る (to sit) is wago, so it takes お. The stem is <ruby>座<rt>すわ</rt></ruby>り. <ruby>下<rt>くだ</rt></ruby>さい elevates the imperative.
- ご<ruby>覧<rt>らん</rt></ruby><ruby>下<rt>くだ</rt></ruby>さい: <ruby>覧<rt>らん</rt></ruby>する (to view) is kango, so it takes ご. The stem/noun is <ruby>覧<rt>らん</rt></ruby>. <ruby>下<rt>くだ</rt></ruby>さい elevates the imperative.
- ご<ruby>利用<rt>りよう</rt></ruby><ruby>下<rt>くだ</rt></ruby>さい: <ruby>利用<rt>りよう</rt></ruby>する (to use) is kango, so it takes ご. The stem/noun is <ruby>利用<rt>りよう</rt></ruby>.
- お<ruby>召<rt>め</rt></ruby>し<ruby>上<rt>あ</rt></ruby>がり<ruby>下<rt>くだ</rt></ruby>さい: <ruby>召<rt>め</rt></ruby>し<ruby>上<rt>あ</rt></ruby>がる (to eat — respectful verb) is wago, so it takes お. The stem is <ruby>召<rt>め</rt></ruby>し<ruby>上<rt>あ</rt></ruby>がり. This is already a respectful verb; adding お/ご〜<ruby>下<rt>くだ</rt></ruby>さい makes it ultra-respectful.
- The subject of these sentences is always the listener (you), but it is never stated. The grammar of <ruby>尊敬語<rt>そんけいご</rt></ruby> implies "you, the honored person."
- Without the prefix: <ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい is ungrammatical. The お/ご is required for the respectful frame.

| お<ruby>掛<rt>か</rt></ruby>け<ruby>下<rt>くだ</rt></ruby>さい。 | Please sit down [hanging / seated position]. |
|---|---|
| | *<ruby>Context<rt>こんてくすと</rt></ruby>: <ruby>掛<rt>か</rt></ruby>ける (to hang / to set) in お<ruby>掛<rt>か</rt></ruby>けになる means "to seat oneself." お<ruby>掛<rt>か</rt></ruby>け<ruby>下<rt>くだ</rt></ruby>さい is a slightly more formal variant of お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい in some service contexts.* |

!!! warning "The wago/kango boundary"
    Most errors involve applying お to kango or ご to wago. If the word can be written in pure hiragana or has native <ruby>Japanese<rt>じゃぱにーず</rt></ruby> etymology, it likely takes お. If it is a two-<ruby>kanji<rt>かんじ</rt></ruby> compound or clearly <ruby>Sino<rt>しの</rt></ruby>-<ruby>Japanese<rt>じゃぱにーず</rt></ruby>, it takes ご.

!!! warning "Some words are lexicalized chunks"
    ご<ruby>覧<rt>らん</rt></ruby>, ご<ruby>注意<rt>ちゅうい</rt></ruby>, お<ruby>待<rt>ま</rt></ruby>ち are so common they should be learned as complete units rather than assembled from rules each time.

!!! warning "Do not use お/ご〜<ruby>下<rt>くだ</rt></ruby>さい for your own actions"
    This is <ruby>尊敬語<rt>そんけいご</rt></ruby> (respectful), not <ruby>謙譲語<rt>けんじょうご</rt></ruby> (humble). It points upward at the listener. Using it for yourself (お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい when you are the one sitting) is a social-direction error.

## <ruby>Japanese<rt>じゃぱにーず</rt></ruby> <ruby>Mindset<rt>まいんどせっと</rt></ruby> {#japanese-mindset}

<ruby>Japanese<rt>じゃぱにーず</rt></ruby> keigo is not "polite <ruby>Japanese<rt>じゃぱにーず</rt></ruby>" — it is a **social positioning system**. お/ご〜<ruby>下<rt>くだ</rt></ruby>さい does more than soften a command. It restructures the relationship between speaker and listener.

When a hotel clerk says お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい, they are not merely saying "Sit down." They are saying: "You are a guest of high standing, and your act of sitting is an honor. I am grateful that you will do it." The grammar encodes an entire service philosophy.

- **Elevation as hospitality.** The prefix お/ご lifts the listener's action out of the everyday. Sitting becomes "being seated." Looking becomes "viewing." Eating becomes "partaking." The everyday is ceremonialized.
- **Request as gift.** <ruby>下<rt>くだ</rt></ruby>さい derives from <ruby>下<rt>くだ</rt></ruby>さる (to give from above). The listener is framed as giving the action to the speaker. Even when the listener is clearly following an instruction, the grammar treats them as bestowing a favor.
- **The wago/kango split as cultural taxonomy.** The fact that native and <ruby>Sino<rt>しの</rt></ruby>-<ruby>Japanese<rt>じゃぱにーず</rt></ruby> vocabulary take different prefixes reveals how deeply <ruby>Japanese<rt>じゃぱにーず</rt></ruby> grammar encodes the origin of words. The particle system respects etymology.

## <ruby>Nuance<rt>にゅあんす</rt></ruby> <ruby>Comparison<rt>こんぱりそん</rt></ruby> {#nuance-comparison}

| <ruby>Grammar<rt>ぐらまー</rt></ruby> <ruby>Point<rt>ぽいんと</rt></ruby> | <ruby>Core<rt>こあ</rt></ruby> <ruby>Meaning<rt>いみ</rt></ruby> | <ruby>Social<rt>そーしゃる</rt></ruby> <ruby>Direction<rt>でぃれくしょん</rt></ruby> | <ruby>Register<rt>れじすたー</rt></ruby> | <ruby>Trigger<rt>とりがー</rt></ruby> <ruby>Context<rt>こんてくすと</rt></ruby> | JLPT |
|---|---|---|---|---|---|
| お/ご〜<ruby>下<rt>くだ</rt></ruby>さい | "Please do X" [respectful] | Elevates listener | Formal, service, upward | Speaking to guests, clients, superiors | N4 |
| 〜て<ruby>下<rt>くだ</rt></ruby>さい | "Please do X" [neutral] | Neutral request | Polite, general | Peers, general requests | N5 |
| 〜<ruby>なさい<rt>なさい</rt></ruby> | "Do X" [directive] | <ruby>Speaker<rt>すぴーかー</rt></ruby> directs listener | Authority to subordinate | <ruby>Parent<rt>ぱれんと</rt></ruby> to child, <ruby>teacher<rt>てぃーちゃー</rt></ruby> to <ruby>student<rt>すとゅーでんと</rt></ruby> | N4 |
| お/ご〜<ruby>する<rt>する</rt></ruby> | "I do X" [humble] | Lowers <ruby>speaker<rt>すぴーかー</rt></ruby> | Humble | Describing one's own actions to superiors | N4 |
| お/ご〜いただけますか | "Could you do X?" [humble request] | Humble request for listener's action | Ultra-polite | Asking superiors for permission/action | N3 |
| お/ご〜<ruby>なさい<rt>なさい</rt></ruby> | "Do X" [respectful directive] | Elevated directive | Formal, slightly old-fashioned | Formal instruction, written | N4 |
| 〜ていただけますか | "Could I receive X?" [humble] | <ruby>Receiving<rt>りしーびんぐ</rt></ruby> frame | Polite | Requesting someone do something for you | N3 |

**お/ご〜<ruby>下<rt>くだ</rt></ruby>さい vs 〜て<ruby>下<rt>くだ</rt></ruby>さい:** Both mean "please do." The difference is elevation. <ruby>座<rt>すわ</rt></ruby>って<ruby>下<rt>くだ</rt></ruby>さい is a polite request to a peer. お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい is a respectful request to a guest. Using 〜て<ruby>下<rt>くだ</rt></ruby>さい with a hotel guest sounds slightly brusque; using お/ご〜<ruby>下<rt>くだ</rt></ruby>さい with a close friend sounds oddly ceremonial.

**お/ご〜<ruby>下<rt>くだ</rt></ruby>さい vs お/ご〜<ruby>する<rt>する</rt></ruby>:** These are opposite directions. お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい = "You, please sit" (respectful). お<ruby>座<rt>すわ</rt></ruby>り<ruby>します<rt>します</rt></ruby> = "I will sit" (humble). The prefix is the same; the social direction is reversed by the main verb (<ruby>下<rt>くだ</rt></ruby>さい vs <ruby>する<rt>する</rt></ruby>).

## <ruby>Alternatives<rt>おるたなてぃぶす</rt></ruby> {#alternatives}

### 〜て<ruby>下<rt>くだ</rt></ruby>さい (neutral request)

The standard polite request. No elevation.

| | |
|---|---|
| **<ruby>Example<rt>いぐざんぷる</rt></ruby>** | <ruby>座<rt>すわ</rt></ruby>って<ruby>下<rt>くだ</rt></ruby>さい。 |
| **<ruby>English<rt>いんぐりっしゅ</rt></ruby>** | "Please sit." |
| **<ruby>Intent<rt>いんてんと</rt></ruby>** | Polite but neutral. The listener is a peer or the context is casual. |
| **vs. お/ご〜<ruby>下<rt>くだ</rt></ruby>さい** | Removes the ceremonial elevation. Appropriate for colleagues, friends, or everyday service. |

!!! tip "When to use this instead"
    Use 〜て<ruby>下<rt>くだ</rt></ruby>さい when speaking to peers, subordinates in casual contexts, or when the situation does not require service-industry formality.

### 〜<ruby>なさい<rt>なさい</rt></ruby> (directive)

Authority speaking downward. Not polite in the deferential sense — it is a command wrapped in grammar.

| | |
|---|---|
| **<ruby>Example<rt>いぐざんぷる</rt></ruby>** | <ruby>座<rt>すわ</rt></ruby>り<ruby>なさい<rt>なさい</rt></ruby>。 |
| **<ruby>English<rt>いんぐりっしゅ</rt></ruby>** | "Sit down." |
| **<ruby>Intent<rt>いんてんと</rt></ruby>** | Directive. The <ruby>speaker<rt>すぴーかー</rt></ruby> has authority. |
| **vs. お/ご〜<ruby>下<rt>くだ</rt></ruby>さい** | <ruby>なさい<rt>なさい</rt></ruby> commands; お/ご〜<ruby>下<rt>くだ</rt></ruby>さい requests. Never use <ruby>なさい<rt>なさい</rt></ruby> with guests or superiors. |

### お/ご〜いただけますか (humble request)

Asks the listener to "give" the action in an even more humble frame.

| | |
|---|---|
| **<ruby>Example<rt>いぐざんぷる</rt></ruby>** | お<ruby>座<rt>すわ</rt></ruby>りいただけますか。 |
| **<ruby>English<rt>いんぐりっしゅ</rt></ruby>** | "Could you please be seated?" |
| **<ruby>Intent<rt>いんてんと</rt></ruby>** | The <ruby>speaker<rt>すぴーかー</rt></ruby> positions themselves as receiving the listener's action. More humble than お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい. |
| **vs. お/ご〜<ruby>下<rt>くだ</rt></ruby>さい** | いただけますか adds a <ruby>receiving<rt>りしーびんぐ</rt></ruby> frame (<ruby>謙譲語<rt>けんじょうご</rt></ruby> layer on top of <ruby>尊敬語<rt>そんけいご</rt></ruby>). Appropriate for very high-status listeners or formal written requests. |

### お/ご〜<ruby>なさい<rt>なさい</rt></ruby> (respectful directive)

A middle ground between request and directive. Slightly old-fashioned.

| | |
|---|---|
| **<ruby>Example<rt>いぐざんぷる</rt></ruby>** | お<ruby>休<rt>やす</rt></ruby>み<ruby>なさい<rt>なさい</rt></ruby>。 |
| **<ruby>English<rt>いんぐりっしゅ</rt></ruby>** | "Do rest." / "Good night." |
| **<ruby>Intent<rt>いんてんと</rt></ruby>** | Elevated directive. Common in set phrases (お<ruby>休<rt>やす</rt></ruby>み<ruby>なさい<rt>なさい</rt></ruby> = good night). Less common for spontaneous requests. |
| **vs. お/ご〜<ruby>下<rt>くだ</rt></ruby>さい** | <ruby>なさい<rt>なさい</rt></ruby> is more decisive; <ruby>下<rt>くだ</rt></ruby>さい is more inviting. |

## <ruby>Register<rt>れじすたー</rt></ruby> and <ruby>Formality<rt>ふぉーまりてぃ</rt></ruby> {#register-and-formality}

The respectful request is formal by definition, but its warmth can be modulated.

=== ":material-chat-outline: <ruby>Casual<rt>かじゅある</rt></ruby> (inappropriate)"
    お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい。

    *"Please be seated."*

    **Shift:** Used with a close friend. Sounds mock-formal or sarcastic unless the context is genuinely ceremonial.

=== ":material-tie: <ruby>Basic<rt>べーしっく</rt></ruby> <ruby>Polite<rt>ぽらいと</rt></ruby> (service standard)"
    どうぞ、お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい。

    *"Please, be seated."*

    **Shift:** <ruby>Standard<rt>すたんだーど</rt></ruby> service-industry warmth. The どうぞ adds invitation; the お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい provides the elevation.

=== ":material-briefcase-outline: <ruby>Extra<rt>えくすとら</rt></ruby> <ruby>Polite<rt>ぽらいと</rt></ruby> (business)"
    ご<ruby>確認<rt>かくにん</rt></ruby>のほど、よろしくお<ruby>願<rt>ねが</rt></ruby>いいたします。

    *"We kindly ask that you please confirm."*

    **Shift:** ご<ruby>確認<rt>かくにん</rt></ruby> + のほど (formal periphrasis) + よろしくお<ruby>願<rt>ねが</rt></ruby>いいたします (humble closing). The core pattern ご<ruby>確認<rt>かくにん</rt></ruby><ruby>下<rt>くだ</rt></ruby>さい is embedded in a more elaborate formal structure.

=== ":material-crown: <ruby>Ultra<rt>うるとら</rt></ruby>-<ruby>Formal<rt>ふぉーまる</rt></ruby> (high-end hospitality)"
    どうぞ、お<ruby>召<rt>め</rt></ruby>し<ruby>上<rt>あ</rt></ruby>がりくださいませ。

    *"Please, do partake."*

    **Shift:** <ruby>下<rt>くだ</rt></ruby>さい + ませ adds historical/formal flourish. Found in luxury ryokan, traditional restaurants, and theatrical speech.

## <ruby>Common<rt>こもん</rt></ruby> <ruby>Mistakes<rt>みすていくす</rt></ruby> {#common-mistakes}

### 1. Using お with kango or ご with wago

!!! failure "<ruby>Incorrect<rt>いんこれくと</rt></ruby> prefix"
    お<ruby>利用<rt>りよう</rt></ruby><ruby>下<rt>くだ</rt></ruby>さい。

    ~<ruby>利用<rt>りよう</rt></ruby> is kango (<ruby>Sino<rt>しの</rt></ruby>-<ruby>Japanese<rt>じゃぱにーず</rt></ruby>). It requires ご, not お.~

!!! success "<ruby>Correct<rt>これくと</rt></ruby>"
    ご<ruby>利用<rt>りよう</rt></ruby><ruby>下<rt>くだ</rt></ruby>さい。

!!! failure "<ruby>Incorrect<rt>いんこれくと</rt></ruby> prefix"
    ご<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい。

    ~<ruby>座<rt>すわ</rt></ruby>る is wago (native <ruby>Japanese<rt>じゃぱにーず</rt></ruby>). It requires お, not ご.~

!!! success "<ruby>Correct<rt>これくと</rt></ruby>"
    お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい。

### 2. Confusing <ruby>尊敬語<rt>そんけいご</rt></ruby> (respectful) with <ruby>謙譲語<rt>けんじょうご</rt></ruby> (humble)

!!! failure "<ruby>Social<rt>そーしゃる</rt></ruby> direction error"
    お<ruby>座<rt>すわ</rt></ruby>り<ruby>します<rt>します</rt></ruby> → said to a guest as an invitation to sit.

    ~This means "I will sit" (humble). You have just announced that you, the staff member, will be sitting. The guest is left standing.~

!!! success "<ruby>Correct<rt>これくと</rt></ruby>"
    お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい。 = "Please sit" (respectful, to guest).
    お<ruby>座<rt>すわ</rt></ruby>り<ruby>します<rt>します</rt></ruby>。 = "I will sit" (humble, about self).

### 3. Using お/ご〜<ruby>下<rt>くだ</rt></ruby>さい in casual peer contexts

!!! failure "<ruby>Register<rt>れじすたー</rt></ruby> mismatch"
    <ruby>友達<rt>ともだち</rt></ruby>に「お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい」

    *"Please be seated, friend."* — Sounds mock-formal or sarcastic. <ruby>Friends<rt>ふれんどす</rt></ruby> say <ruby>座<rt>すわ</rt></ruby>って or <ruby>座<rt>すわ</rt></ruby>ってよ.

!!! success "Appropriate"
    <ruby>座<rt>すわ</rt></ruby>って。 / <ruby>座<rt>すわ</rt></ruby>って<ruby>下<rt>くだ</rt></ruby>さい。 (neutral/casual polite)

### 4. Using polite ます stem instead of plain stem

!!! failure "<ruby>Stem<rt>すてむ</rt></ruby> error"
    お<ruby>座<rt>すわ</rt></ruby>り<ruby>ますください<rt>ますください</rt></ruby>。

    ~Ungrammatical. The stem is <ruby>座<rt>すわ</rt></ruby>り, not <ruby>座<rt>すわ</rt></ruby>ります.~

!!! success "<ruby>Correct<rt>これくと</rt></ruby>"
    お<ruby>座<rt>すわ</rt></ruby>り<ruby>下<rt>くだ</rt></ruby>さい。
