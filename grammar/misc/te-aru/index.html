    Intentional Preparation That Remains¶

    〜てある — "Has Been Done (Deliberately) and Is Still in That State"¶

    JLPT Level¶

    JLPT Level — Upper N4 / Lower N3

    The form 〜てある appears at N4, but the nuance — intentional setup and resulting state — is more N3-level.

    Tier Pattern Description Example
    N4 — Foundation 〜ている ongoing / resulting state (neutral) ドアが開あいている。"The door is open."
    N4 — Core ← You are here 〜てある intentional result state left by someone ドアが開あけてある。"The door has been opened (on purpose)."
    N3 — Extended omission of agent focus only on result, not who did it 資料しりょうは用意よういしてあります。
    N2 — Awareness contrast nuance mastery subtle difference from 〜ている in context [Coming soon]

    Quick Summary¶

    At a glance

    • 〜てある describes a state that exists because someone deliberately did something earlier — and that action's effect or purpose remains relevant now.
    • It always implies an agent — someone intentionally performed the action, and the resulting state is there for a reason.
    • In the reference sentence — そのことはもう社長しゃちょうにお話はなししてあります — the speaker is saying "I have already spoken to the company president about that matter (and the president is now informed — the information is in place, deliberately)."
    • 〜てある uses transitive verbs and marks the object with が (not を) — the object is the thing that has been acted upon and now sits in its prepared state.
    • This pattern is distinct from 〜ている on change verbs (which describes a state without emphasising intention) and from past tense 〜た (which only says the action happened, with no comment on the resulting state).

    The Core Distinction — 〜てある vs. 〜ている vs. 〜た¶

    Three Ways to Talk About What Happened — and What Remains

    Japanese offers three different ways to talk about a completed action. They are not interchangeable.

    Form What It Describes Agent Implied? State Implied? Example
    〜た (past) The action happened. Sometimes (context) No 話はなした。 "I spoke."
    〜ている (resultative) The result of a change is in effect. No — just the state Yes 開あいている。 "It is open."
    〜てある (prepared state) Someone deliberately did something earlier, and the prepared state remains relevant. Yes — strongly Yes — the point 話はなしてある。 "I've spoken (and the information is in place)."

    The reference sentence uses 〜てある precisely because the speaker wants to communicate: I did this deliberately, earlier, and the result — the president now knowing — is the relevant current state. The matter is handled.

    The Reference Sentence — Full Breakdown¶

    The Sentence¶

    そのことはもう社長しゃちょうにお話はなししてあります。

    Natural Translation¶

    "I have already spoken to the company president about that matter." Or more precisely: "That matter has been discussed with the president (by me, deliberately, and the president is now informed)."

    Literal Translation¶

    "That matter, already, president-to speaking-has-been-done-and-remains."

    Component Breakdown¶

    Component Form Function
    そのこと Noun phrase "That matter" — the topic / the thing that has been handled
    は Topic particle Frames そのこと as the discourse topic
    もう Adverb "Already" — the action was completed earlier
    社長しゃちょうに Noun + に "To the company president" — the target of the speaking
    お話はなし Honorific verb stem 話はなす → お話はなしする (humble form) — the speaker lowers themselves relative to the president
    して Te-form of する The te-form connects to ある
    あります Polite form of ある てある in polite register — "exists in the deliberately-done state"

    Why 〜てある and Not 〜た or 〜ている?¶

    Form What It Would Say Why It's Different
    話はなした "I spoke." Only says the action happened. No comment on whether the information is still in place, whether it was deliberate, or whether it matters now.
    話はなしている "I am speaking" (progressive) or possibly resultative but odd with 話はなす For 話はなす, 〜ている sounds progressive ("am speaking right now"), not resultative. It doesn't communicate "the speaking is done and the information is in place."
    話はなしてある "I have spoken (deliberately, and the president now knows)." The right choice. Communicates intentionality, completion, and continuing relevance.

    How 〜てある Works — The Full Mechanism¶

    The Three Components of 〜てある¶

    1. An agent (usually the speaker) deliberately performs an action at some earlier time.
    2. That action is completed.
    3. The resulting state — the outcome of the action — persists with ongoing relevance. The state exists on purpose, for a reason.

    In the reference sentence: - Agent: The speaker (implied — humble form お話はなしする marks the speaker as the actor). - Action: Speaking to the president about the matter. - Resulting state: The president is now informed. The information is in place. The matter is handled. - Ongoing relevance: The speaker is telling someone that the matter has been taken care of — no further action is needed, or the groundwork is laid.

    The Grammar — Transitive Verbs + が¶

    〜てある almost exclusively uses transitive verbs (verbs that take a direct object). But in the 〜てある construction, the object is marked with が (not を), because the focus shifts from the action to the resulting state of the object.

    Standard (Action) 〜てある (Prepared State)
    窓まどを開あける 窓まどが開あけてある
    "I open the window." (action, を) "The window has been opened (by someone, deliberately, and is open)." (state, が)
    机つくえの上うえに本ほんを置おく 机つくえの上うえに本ほんが置おいてある
    "I put a book on the desk." (action, を) "A book has been placed on the desk (and is there)." (state, が)

    In the reference sentence, the object of the action is そのこと — but it's marked with は (topic) rather than が (subject). The underlying structure is そのことが社長しゃちょうに話はなしてある — "that matter has been spoken to the president about." The は replaces が for topic prominence, but the logic is the same: the matter is in the "deliberately handled" state.

    Nuance Comparison¶

    Pattern Meaning Nuance Example JLPT
    〜ている state / ongoing neutral ドアが開あいている N5
    〜てある result of intentional action prepared / done ドアが開あけてある N4
    〜ておく do in advance preparation action ドアを開あけておく N4

    Micro Contrast¶

    Sentence Meaning Difference
    ドアが開あいている The door is open state
    ドアが開あけてある The door has been opened (on purpose) intentional result

    〜てある vs. 〜ている — The Critical Contrast¶

    The closest lookalike is the resultative 〜ている on change verbs. Both describe a resulting state. The difference is intention and agent.

    〜ている (Resultative) 〜てある
    Verb type Intransitive change verbs Transitive action verbs
    Agent Not implied — the state just exists Strongly implied — someone did this on purpose
    Nuance "It is in this state." "It has been deliberately put in this state and is here for a reason."
    Example 窓まどが開あいている。 窓まどが開あけてある。
    Meaning "The window is open." (state; no comment on why or who) "The window has been opened (by someone, on purpose, for a reason — to air the room, etc.)."

    Transitive verb + てある; Intransitive verb + ている

    The pair is almost always: - Intransitive change verb + ている → state without agent - Transitive verb + てある → state with implied agent

    Intransitive + ている Transitive + てある
    開あく → 開あいている (is open) 開あける → 開あけてある (has been opened)
    閉しまる → 閉しまっている (is closed) 閉しめる → 閉しめてある (has been closed)
    落おちる → 落おちている (has fallen) 落おとす → 落おとしてある (has been dropped / placed deliberately)
    沸わく → 沸わいている (has boiled / is boiling) 沸わかす → 沸わかしてある (has been boiled / water has been boiled and is ready)

    The Agent Question — Who Did It?¶

    〜てある implies an agent. The identity of that agent is determined by context and the form of the verb.

    Verb Form Agent Example
    Plain transitive Usually the speaker, or someone contextually clear 鍵かぎをかけてある。 "I've locked it." / "Someone has locked it."
    Humble (お〜する) Always the speaker お話はなししてあります。 "I have spoken (humbly)."
    Respectful (お／ご〜になる) Someone the speaker respects (Rare with 〜てある, but possible: お書かきになってある → "the honoured person has written it")
    Passive + てある Someone else did it; the speaker is reporting the state 準備じゅんびがされている。 "Preparations have been made (by someone — I may not know who)."

    In the reference sentence, the humble form お話はなししてある explicitly marks the speaker as the agent. There is no ambiguity — the speaker is saying "I did this, I'm responsible, the president now knows."

    〜てある vs. 〜ておく — Preparation vs. Prepared State¶

    〜ておく also describes doing something in advance for future benefit, but the focus is different.

    〜ておく 〜てある
    Focus The action of preparing The state after preparation
    Temporal frame Look forward: "I'll do this so it's ready." Look back: "I've done this; it's ready now."
    Example 話はなしておく。 "I'll speak (in advance, so it's taken care of)." 話はなしてある。 "I've spoken (and the result is in place)."

    〜ておく describes the act of preparing; 〜てある describes the state of being prepared. They often form a sequence: you 〜ておく (do something in advance); afterwards, the situation is 〜てある (prepared).

    明日あしたのために話はなしておくね。 → もう話はなしてあるから安心あんしんして。 "I'll speak to them for tomorrow." → "I've already spoken to them, so don't worry."
    〜ておく is the plan; 〜てある is the completed preparation.

    Register and Politeness¶

    The reference sentence uses the humble form お話はなししてあります — a combination that carries significant social information.

    そのこと、もう社長しゃちょうに話はなしてあるよ。

    "I've already talked to the boss about that."

    Register: Casual. Plain 話はなしてある with よ for warmth. Appropriate when speaking to a peer or close colleague about having handled something.

    そのことはもう社長しゃちょうに話はなしてあります。

    "I have already spoken to the president about that matter."

    Shift: 話はなしてある → 話はなしてあります (polite). Clean, professional, appropriate for most workplace contexts when speaking to a colleague or superior about work you've done.

    そのことはもう社長しゃちょうにお話はなししてあります。

    "I have already spoken to the company president about that matter (humbly)."

    Shift: 話はなして → お話はなしして (humble お〜する form). The speaker lowers themselves relative to the president while describing their own action. The 〜てある construction remains — the speaker is taking credit for the action (I did it) but framing it humbly (I humbly did it). This combination — humble verb + てある — is common in professional Japanese: you claim responsibility while showing appropriate deference.

    Extended Examples — 〜てある in Context¶

    Deliberate Preparation¶

    Japanese English
    資料しりょうはもう全すべて用意よういしてあります。 All the materials have been prepared (and are ready).
    The preparation is done. The state of readiness is current. The agent (speaker/team) did this deliberately.
    明日あしたの会議かいぎのことは、みんなに連絡れんらくしてあるよ。 I've already let everyone know about tomorrow's meeting.
    The notification is complete. Everyone is informed. The state is in place.
    ホテルhotelはもう予約よやくしてあります。 The hotel has been booked (and the reservation is in place).
    The booking is done. The reservation exists as the prepared state.

    Physical Arrangement — Something Placed or Set¶

    Japanese English
    テーブルの上うえに花はなが飾かざってある。 Flowers have been arranged on the table (and are there now).
    が marks the flowers — they sit in the "deliberately placed" state. Someone arranged them.
    冷蔵庫れいぞうこにビールが冷ひやしてあるよ。 There's beer chilling in the fridge (I put it there on purpose).
    The beer is in the "deliberately cooled" state. The speaker did this for a reason.
    壁かべに地図ちずが貼はってある。 A map has been put up on the wall (and is there).
    Someone stuck it there. The map remains in its placed state.

    Information in Place — Things That Have Been Said or Written¶

    Japanese English
    その件けんはもうメールで伝つたえてあります。 I've already communicated that matter via email.
    The communication is done. The information is in place.
    注意事項ちゅういじこうは掲示板けいじばんに書かいてあります。 The precautions have been written up on the notice board.
    The writing is done; the advisory is posted; anyone can read it.
    彼かれにはもう結果けっかを報告ほうこくしてある。 I've already reported the results to him.
    He knows. The information has been delivered. The state is handled.

    The "For a Reason" Nuance¶

    〜てある always carries an implication of purpose. The state exists because someone wanted it that way — it serves a function. This is the most important nuance that distinguishes it from the neutral resultative 〜ている.

    窓まどが開あいている。 The window is open. (Maybe the wind blew it. Maybe someone forgot. No comment on purpose.)
    窓まどが開あけてある。 The window has been opened (deliberately — to air the room, to let in fresh air).

    The second sentence implies a reason. The listener can infer it (ventilation) or ask. The first sentence is just a description of state.

    In the reference sentence, 話はなしてある implies purpose: the president has been informed so that the matter is handled, so that no further action is needed, or so that the groundwork is laid for whatever comes next. The speaker didn't just speak — they spoke for a reason, and the result is in place.

    〜てある in the Passive — 〜されている¶

    When the agent is unknown, unimportant, or deliberately obscured, Japanese often uses the passive form + ている or てある to describe a prepared state without specifying who did it.

    Form Example Agent
    Plain 〜てある 準備じゅんびしてある。 Speaker (or known from context)
    Passive + 〜ている 準備じゅんびがされている。 Unspecified — "preparations have been made"
    Passive + 〜てある 準備じゅんびがされてある。 Less common, but possible — strongly emphasises the state as deliberately arranged by someone
    会場かいじょうにはもう椅子いすが並ならべてあった。 The chairs had already been arranged at the venue (by someone).
    Past form 〜てあった — at the reference time, the deliberate arrangement was already in place.

    Common Mistakes¶

    1. Using を instead of が with 〜てある¶

    Incorrect

    窓まどを開あけてある。

    ~The 〜てある construction shifts focus from the action to the resulting state. The object of the action becomes the subject (が-marked) of the state description. を is the action particle; it clashes with the state framing.~

    Correct

    窓まどが開あけてある。

    ~が marks the window as the thing existing in the deliberately-opened state.~

    Exception: Topic は

    窓まどは開あけてある is also correct — は replaces が as the topic marker. The underlying structure is still が, but は overlays it for discourse purposes.

    2. Using intransitive verbs with 〜てある¶

    Incorrect

    窓まどが開あいてある。

    ~開ひらく is intransitive ("to open" on its own). Intransitive verbs pair with 〜ている for resultative states (開ひらいている = "is open"). 〜てある requires a transitive verb — someone has to do the opening.~

    Correct

    窓まどが開あけてある。

    ~開あける is transitive ("to open something"). 開あけてある = "has been opened (by someone, deliberately)."~

    3. Using 〜てある for spontaneous or natural states¶

    Incorrect

    木きが倒たおしてある。

    ~倒たおす is transitive ("to knock down"). Unless someone deliberately knocked the tree down, this is wrong. Natural occurrences use intransitive + ている.~

    Correct

    木きが倒たおれている。

    ~倒たおれる is intransitive ("to fall over"). 倒たおれている = "has fallen / is lying fallen." No agent implied.~

    Japanese Mindset — Responsibility and Readiness¶

    〜てある reflects a Japanese cultural emphasis on responsibility and preparedness. When you say 話はなしてあります, you are not just reporting an action — you are taking responsibility for the resulting state. The president knows because you made sure of it. The matter is handled because you handled it.

    This is distinct from the English present perfect ("I have spoken"), which reports the action but doesn't necessarily foreground the prepared state. The Japanese 〜てある says: the state is in place, I am the one who put it there, and it's there for a reason.

    • Accountability is built into the grammar. Using 〜てある claims the action and its outcome. In professional contexts, this is crucial — you are telling someone that something is taken care of, and you are the one who took care of it.
    • The が-marked object tells you what's been handled. The thing marked with が in a 〜てある sentence — the window, the document, the message — is the thing whose state has been deliberately arranged. Framing it as the grammatical subject emphasises its readiness.
    • 〜てある closes a loop. When you say 話はなしてあります, the implicit message is "this is done; you don't need to worry about it; we can move forward." It's a closure marker as much as a state description.

    Quick Reference — 〜てある Patterns¶

    Pattern Meaning Example
    Transitive verb + てある Deliberate action done; state in place 話はなしてある "I've spoken (and it's handled)"
    Object が + transitive てある The object is in the deliberately-arranged state 窓まどが開あけてある "The window has been opened"
    Intransitive verb + ている Natural state; no agent implied 窓まどが開あいている "The window is open"
    Humble お／ご〜してある Speaker humbly did the action; result in place お話はなししてあります "I have spoken (humbly, and it's handled)"
    Passive + てある／られている Agent unspecified; state arranged by someone 準備じゅんびがされている "Preparations have been made"
    〜てあった (past) State was in place at a past reference time 準備じゅんびしてあった "It had been prepared (when I checked)"

    Pattern Expansion¶

    Common Builds¶

    • 〜てある → result state
    • 〜ておく → preparation
    • 〜てある + から → justification

    Upgrade / Evolution Path¶

    • Can recognize 〜てある as “already done”
    • Still rely on translation
    • Begin distinguishing from 〜ている
    • Eventually feel “intentional result” naturally

    Collocation Notes¶

    Collocation Meaning
    準備じゅんびしてある prepared
    予約よやくしてある reserved
    説明せつめいしてある explained
    伝つたえてある informed

    Hidden Points¶

    • Always implies intentional action by someone
    • The actor is usually omitted
    • Focus is on state/result
    • Common in workplace and coordination contexts
    • Often used to reassure or confirm completion

    Same Aspect Family¶

    • 〜ている (resultative) — non-agentive state: "is in the state of having become X"
    • 〜ておく — preparatory action: "do X in advance so it's ready"
    • 〜た (past) — simple completed action: "did X" (no state implied)

    Related Constructions¶

    • 〜てみる — "try doing X and see the result" (also uses te-form + auxiliary, different meaning)
    • 〜ことになっている — "it has been arranged that" — describes an institutional state, not a physical one
    • Passive + ている — "has been done (by someone unspecified)" — agent obscured

    Contrast With¶

    • Intransitive + ている vs. Transitive + てある — the core distinction for natural vs. deliberate states
    • 〜た vs. 〜てある — completed action vs. completed action with ongoing state
